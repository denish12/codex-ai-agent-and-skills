# Skill: UI & A11y Smoke Review

## Проверить
- Есть loading/empty/error состояния по UX Spec
- Фокус-стили видимы, можно пользоваться клавиатурой
- Формы: label/aria, сообщения об ошибках доступны
- Нет очевидных регрессий контента/CTA

## Выход
- Findings + рекомендации
