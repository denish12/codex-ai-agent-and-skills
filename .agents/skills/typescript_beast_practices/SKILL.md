# Skill: TypeScript Beast Practices

## Цель
Типобезопасность контрактов и рефакторинг без страха.

## Правила
- strict по возможности
- DTO/Contracts типизируются и валидируются на границах
- Не использовать any без явного обоснования
- Общие типы и утилиты — в одном месте, без циклических зависимостей
