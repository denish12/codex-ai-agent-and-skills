# Skill: Styling (CSS/Less/Sass/Tailwind)

## Правила
- Tailwind по умолчанию (если проект так устроен)
- Если используется CSS/Less/Sass — единые naming-конвенции и структура
- UI должен соответствовать UX/a11y: контраст, фокус-стили, состояния
- Избегать “случайных” глобальных стилей, которые ломают соседние компоненты
