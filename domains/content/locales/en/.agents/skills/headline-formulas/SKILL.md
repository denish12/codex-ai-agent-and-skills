# Headline Formulas

## When to Use
- When writing **post headlines**, articles, email subject lines, ads.
- When creating **hooks** for reels, videos, carousels (first frame / first seconds).
- When **optimizing existing** headlines — low CTR, low open rate.
- As part of **`$content-brief`** — "Headline / Hook" section.

## Input

| Field | Required | Description |
|------|:-----------:|----------|
| Topic / key insight | ✅ | What the content is about — main idea or fact |
| Target audience persona | ✅ | From `$audience-analysis` — language, pains, triggers |
| Platform | ✅ | Determines character limits and style |
| Content type | ✅ | Post / article / email / reel / carousel / landing page / ad |
| Funnel stage | ✅ | TOFU / MOFU / BOFU — determines the approach |
| SEO keyword | ⬚ | If SEO content — main keyword for title |
| Current headline | ⬚ | If optimization — current version and its metrics |

> If persona or platform are not specified — **request**. A headline without a target audience is a headline for nobody.

## Headline Formulas

### Basic (Universal)

| # | Formula | Template | Example | Best for |
|---|---------|--------|--------|-----------------|
| 1 | **Number + Result** | "[N] ways to [achieve result]" | "7 ways to double your email conversion" | Articles, carousels, listicles |
| 2 | **How-to** | "How to [achieve goal] without [problem]" | "How to get your first 1000 followers with zero budget" | Educational, TOFU/MOFU |
| 3 | **Question + Pain** | "Why [problem]? (and how to fix it)" | "Why your posts aren't getting reach?" | Posts, emails, nurturing |
| 4 | **Contradiction** | "[Opinion] — is actually [fact]" | "Everyone says to post daily — here is why it doesn't work" | Posts, viral, TOFU |

### Advanced (High CTR)

| # | Formula | Template | Example | Best for |
|---|---------|--------|--------|-----------------|
| 5 | **4U** | Urgent + Unique + Ultra-specific + Useful | "Before Friday: 3 steps to your first sale via Telegram" | Promo, BOFU, email |
| 6 | **Before/After** | "[Before] → [After] in [time/action]" | "0 followers → 10K in 3 months: what worked" | Case studies, MOFU |
| 7 | **Social Proof** | "[N people/companies] already [result]" | "500+ marketers already use this strategy" | MOFU/BOFU, promo |
| 8 | **Negative / Fear** | "[N] mistakes that [kill result]" | "5 mistakes that are killing your email open rate" | Educational, TOFU |
| 9 | **Curiosity Gap** | "[Intrigue] — [hint without revealing]" | "We found what marketers have ignored for years" | TOFU, viral |
| 10 | **Specificity** | "[Exact number] + [specific result] + [condition]" | "How we got 847 leads for $0.12 each through one carousel" | Case studies, MOFU |

### Specialized

| # | Formula | Template | Example | Best for |
|---|---------|--------|--------|-----------------|
| 11 | **Celebrity / Authority** | "[Authority] recommends [method]" | "Gary V's method for creating 30 posts from one video" | TOFU, educational |
| 12 | **This vs That** | "[A] vs [B]: what works better for [context]" | "Reels vs Carousels: what gets more reach in 2026" | Comparisons, MOFU |
| 13 | **Secret / Insider** | "[Secret/trick] that [nobody talks about/few know]" | "A copywriting trick agencies keep silent about" | TOFU, curiosity |
| 14 | **Challenge** | "[Result] in [time]: [challenge]" | "30 reels in 30 days: what happened to the account" | Series, engagement |
| 15 | **Direct Benefit** | "[Get/Learn/Discover] [result] [right now]" | "Get a ready-made content plan for April — free" | BOFU, lead magnets |

## Formulas by Funnel Stage

| Stage | Headline Goal | Best Formulas | Style |
|------|---------------|----------------|-------|
| **TOFU** | Attract attention, broad reach | #4 Contradiction, #8 Negative, #9 Curiosity, #13 Secret, #14 Challenge | Provocative, intriguing, viral |
| **MOFU** | Deepen interest, build trust | #2 How-to, #3 Question+Pain, #6 Before/After, #7 Social Proof, #10 Specificity, #12 This vs That | Expert, specific, data-driven |
| **BOFU** | Convert | #5 4U, #7 Social Proof, #15 Direct Benefit | Direct, with urgency, value-driven |

## Power Words

Enhancing words that increase headline CTR:

| Category | Words |
|-----------|-------|
| **Value** | free, ready-made, step-by-step, proven, guaranteed |
| **Urgency** | now, today, by [date], last chance, before it's too late |
| **Curiosity** | secret, unexpected, hidden, little-known, actually |
| **Negative** | mistake, kills, losing, never, stop, enough |
| **Emotion** | incredible, shocking, amazing, powerful |
| **Specificity** | exact, step-by-step, specific, [number], in [time] |
| **Exclusive** | only for, for the first time, insider, private |

> Use 1-2 Power Words per headline. More = clickbait.

## Platform Adaptation

| Platform | Character Limit | Style | Features |
|-----------|---------------|-------|-------------|
| **Instagram (caption hook)** | First 125 chars visible without "more" | Short, intriguing, emotional | Hook = first line. Emojis allowed. Caps Lock for emphasis |
| **Instagram (carousel cover)** | 5-8 words on cover | Large text, contrasting | Readability > creativity. Single hook |
| **Telegram** | First 2-3 lines visible in preview | Informative, specific | No clickbait — Telegram audience is unforgiving |
| **LinkedIn** | First 150 chars before "...see more" | Professional, expert | Start with an insight or number. No emojis |
| **Email subject** | ≤ 50 chars (mobile) | Personal, curiosity or value | Preheader complements, doesn't repeat |
| **Google Title (SEO)** | ≤ 60 chars | SEO keyword at start + value | Keyword as early as possible |
| **YouTube** | ≤ 70 chars (truncates ~60) | Curiosity + specificity | Caps Lock for 1-2 words, numbers, emotion |
| **Twitter/X** | ≤ 280 (shorter = better) | Sharp, provocative, meme-like | No links in text (reduces reach) |
| **Ad Copy** | Varies by platform | Direct benefit + urgency | Test 3-5 variants |

## Scorecard — Evaluation

Each headline variant is evaluated on 6 criteria (1-5 points):

| Criterion | 1 (Weak) | 3 (Average) | 5 (Strong) |
|----------|-----------|------------|------------|
| **Clarity** | Unclear what it's about | Understandable if read carefully | Instantly clear |
| **Specificity** | Abstract ("improve marketing") | Hint of specificity | Exact numbers, deadlines, results |
| **Value** | No benefit for reader | Benefit is implied | Explicit benefit in headline |
| **Emotion / Curiosity** | Boring, no reaction | Slightly interesting | Elicits "seriously?!" or "I need to know" |
| **Audience Fit** | For everyone = for nobody | Fits broadly | Speaks the specific persona's language |
| **Platform Fit** | Doesn't fit format | Fits, but not optimal | Perfect for platform (limits, style) |

**Threshold:** ≥ 22 out of 30 — strong headline. 18-21 — revise. < 18 — rewrite.

## Protocol

### Step 1 — Context
1. Get input data: topic, persona, platform, funnel, content type.
2. Determine character limit based on platform.
3. If SEO — set the keyword.

### Step 2 — Formula Selection
1. Based on funnel stage — choose **3-4 suitable formulas** from the table.
2. Consider content type (case study → Before/After, listicle → Number+Result).
3. Consider persona (gen Z → provocation, B2B → expertise).

### Step 3 — Generation
1. For each formula — write **1-2 variants** of the headline.
2. Use **1-2 Power Words** (no more).
3. Check platform character limits.
4. Total: **5-8 variants**.

### Step 4 — Scorecard Evaluation
1. Evaluate each variant on 6 criteria (1-5).
2. Calculate total score.
3. Discard variants < 18 points.

### Step 5 — Final Selection
1. **Primary** — variant with the highest score.
2. **A/B Test** — variant using a different formula (to test the approach, not just wording).
3. Justify the choice.

## Validation (Quality Gate)

Headlines are considered ready if:

- [ ] Generated ≥ 5 variants using ≥ 3 formulas
- [ ] Each variant is within platform limits
- [ ] Each variant evaluated via Scorecard
- [ ] Primary variant ≥ 22 points
- [ ] Variant selected for A/B test (different formula)
- [ ] Headline matches the funnel stage
- [ ] Headline doesn't deceive expectations (no clickbait)
- [ ] Power Words ≤ 2 per headline

## Anti-patterns

| Mistake | Why it's bad | How to do it right |
|--------|-------------|---------------|
| Clickbait without substance | High clicks, but quick exits → ER drops, trust lost | Headline promises — content delivers |
| "Read more" / "New post" | Zero value, nobody clicks | Specific benefit or intrigue |
| 5+ Power Words | Looks like spam: "FREE SECRET INCREDIBLE AMAZING METHOD" | Maximum 1-2 Power Words |
| Headline mismatch with funnel | TOFU audience sees "Buy now" → scroll past | Formula based on funnel stage |
| One variant without A/B | No data for optimization | At least 2 variants with different formulas |
| Long headline on mobile | Truncated, meaning lost | Check platform limits |
| Abstraction without specifics | "Improve your marketing" — boring | Numbers, deadlines, results: "+340% reach in 2 weeks" |
| Headline for yourself, not TA | Uses professional jargon for beginners | Persona language from `$audience-analysis` |
