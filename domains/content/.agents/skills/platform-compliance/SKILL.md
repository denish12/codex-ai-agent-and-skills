# Platform Compliance — Соответствие требованиям платформ

## Когда использовать
- На этапе **Reviewer** — проверка перед `$content-release-gate`.
- При **кросс-постинге** — адаптация контента под другую платформу.
- При **проблемах с охватом** — аудит на алгоритмические риски.
- При **рекламном контенте** — проверка рекламных политик платформы.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Контент для проверки | ✅ | Текст + визуал / видео — что проверяем |
| Платформа | ✅ | Где будет опубликовано |
| Формат контента | ✅ | Пост / карусель / reel / сторис / статья / видео |
| Тип публикации | ✅ | Organic / Paid (реклама) — разные правила |
| Контент-бриф | ⬚ | Для сверки с техническими требованиями из брифа |

> Если платформа не указана — **запросить**. Требования Instagram и LinkedIn — принципиально разные.

## Severity

| Уровень | Определение | Влияние |
|---------|-------------|---------|
| 🔴 **Blocker** | Публикация невозможна или будет удалена/заблокирована | NON-COMPLIANT |
| 🟡 **Risk** | Публикация возможна, но алгоритм снизит охват или есть риск предупреждения | COMPLIANT WITH RISKS |
| 🔵 **Note** | Рекомендация для оптимизации, не нарушение | COMPLIANT |

### Порог прохождения

| Результат | Условие |
|-----------|---------|
| **COMPLIANT** | 0 Blocker + 0 Risk |
| **COMPLIANT WITH RISKS** | 0 Blocker + 1-2 Risk (зафиксированы, пользователь уведомлён) |
| **NON-COMPLIANT** | ≥ 1 Blocker ИЛИ ≥ 3 Risk |

## Реестр требований по платформам

### Instagram

**Технические:**

| Формат | Размер | Ratio | Лимит текста | Другое |
|--------|--------|-------|-------------|--------|
| Feed (фото) | 1080×1080 / 1080×1350 | 1:1 / 4:5 | 2200 символов caption | ≤ 30 хэштегов |
| Carousel | 1080×1080 / 1080×1350 | 1:1 / 4:5 | 2200 символов | ≤ 10 слайдов |
| Reels | 1080×1920 | 9:16 | 2200 символов | 15-90 сек (рекомендуется ≤ 60) |
| Stories | 1080×1920 | 9:16 | — | 15 сек на слайд |

**Правила сообщества:**
- 🔴 Нет: наготы, насилия, hate speech, продажи запрещённых товаров, контента несовершеннолетних
- 🔴 Нет: нарушения авторских прав (музыка, фото, видео)
- 🔴 Маркировка рекламы обязательна (Paid Partnership / пометка «Реклама»)
- 🟡 Engagement bait penalty: «лайкни если...», «напиши + в комменты» — снижает охват
- 🟡 Ссылки в caption не кликабельны — ссылка в bio или стикер в Stories

**Алгоритмические риски:**
- 🟡 Повторное использование контента TikTok с водяным знаком — штраф охвата
- 🟡 Слишком частые публикации (> 2/день) — каннибализация охвата
- 🟡 Хэштеги-бан (проверить актуальный список banned hashtags)
- 🔵 Первые 30-60 минут после публикации критичны для алгоритма — время публикации важно

### LinkedIn

**Технические:**

| Формат | Размер | Ratio | Лимит текста | Другое |
|--------|--------|-------|-------------|--------|
| Post (фото) | 1200×627 / 1080×1080 | 1.91:1 / 1:1 | 3000 символов | — |
| Carousel (PDF) | 1080×1080 / 1920×1080 | 1:1 / 16:9 | — | PDF, ≤ 300 стр |
| Video | 256×144 — 4096×2304 | любой | — | 3 сек — 10 мин |
| Article | — | — | без лимита | Нативный редактор |

**Правила сообщества:**
- 🔴 Нет: hate speech, harassment, spam, фейковые аккаунты, misleading content
- 🔴 Профессиональный контекст — LinkedIn строже к контенту не по теме
- 🟡 Ссылки в посте снижают охват (алгоритм предпочитает нативный контент)
- 🟡 Тегание людей, которые не вовлечены — расценивается как спам

**Алгоритмические риски:**
- 🟡 Внешние ссылки в посте — охват ↓ 40-50%. Ссылку в первый комментарий
- 🟡 Engagement bait: «согласен? лайкни» — penalty
- 🟡 Редактирование поста в первый час — сброс алгоритмического буста
- 🔵 Dwell time важен — длинные посты с форматированием лучше

### Telegram

**Технические:**

| Формат | Размер | Ratio | Лимит текста | Другое |
|--------|--------|-------|-------------|--------|
| Post (текст) | — | — | 4096 символов | Markdown / HTML форматирование |
| Post (фото) | рекомендуется 800×400+ | 2:1 рекомендуется | 1024 символа caption | ≤ 10 фото в альбоме |
| Video | — | любой | 1024 символа caption | ≤ 2 GB |

**Правила:**
- 🔴 Нет: призывов к насилию, экстремизма, детского контента
- 🔴 Маркировка рекламы по законодательству РФ (если для РФ-аудитории)
- 🟡 Спам-жалобы — канал может быть ограничен в рекомендациях
- 🔵 Нет алгоритмической ленты (хронологическая) — время публикации критично

### YouTube

**Технические:**

| Формат | Размер | Ratio | Лимит | Другое |
|--------|--------|-------|-------|--------|
| Video | 1920×1080 рекомендуется | 16:9 | — | Рекомендуется > 8 мин для mid-roll ads |
| Shorts | 1080×1920 | 9:16 | — | ≤ 60 сек |
| Thumbnail | 1280×720 | 16:9 | — | ≤ 2 MB, JPG/PNG |
| Title | — | — | 100 символов (рекомендуется ≤ 70) | — |
| Description | — | — | 5000 символов | Первые 2 строки видны |

**Правила:**
- 🔴 Нет: misleading metadata (clickbait title не соответствует видео), spam, harassment
- 🔴 Нет: повторного контента (reused content penalty — дважды загружен один ролик)
- 🔴 Авторские права: Content ID автоматически определяет музыку и видео
- 🟡 Made for Kids — ограничивает комментарии, уведомления, персонализацию

**Алгоритмические риски:**
- 🟡 CTR thumbnail < 4% — видео получает меньше показов
- 🟡 Retention < 40% — алгоритм перестаёт рекомендовать
- 🔵 Первые 24-48 часов — критичны для алгоритма

### Twitter/X

**Технические:**

| Формат | Размер | Ratio | Лимит текста | Другое |
|--------|--------|-------|-------------|--------|
| Post | — | — | 280 символов (бесплатно) / 25000 (Premium) | — |
| Image | 1600×900 рекомендуется | 16:9 | — | ≤ 4 фото |
| Video | 1920×1200 макс | любой | — | ≤ 2:20 мин (бесплатно) |

**Правила:**
- 🔴 Нет: hate speech, harassment, manipulated media, spam
- 🟡 Ссылки снижают охват (алгоритм предпочитает нативный контент)
- 🟡 Автоматический кросс-постинг (одинаковый текст) — penalty

### TikTok

**Технические:**

| Формат | Размер | Ratio | Лимит текста | Другое |
|--------|--------|-------|-------------|--------|
| Video | 1080×1920 | 9:16 | 2200 символов caption | 15 сек — 10 мин |
| Photo carousel | 1080×1920 | 9:16 | 2200 символов | ≤ 35 фото |

**Правила:**
- 🔴 Нет: dangerous acts, harassment, hate speech, misleading content, minor safety
- 🔴 Нет: нарушения авторских прав (музыка — использовать библиотеку TikTok)
- 🟡 Водяной знак другой платформы — снижение охвата
- 🟡 Контент для 18+ ограничивается в For You

**Алгоритмические риски:**
- 🟡 Первые 3 секунды = hook. Нет hook → scroll away → алгоритм не продвигает
- 🟡 Completion rate критичен — короче = лучше для вовлечения
- 🔵 Trending sounds увеличивают охват

## Протокол

### Шаг 1 — Определение контекста
1. Определить платформу, формат контента, тип (organic/paid).
2. Загрузить требования платформы из реестра.

### Шаг 2 — Техническая проверка
1. Проверить **все технические параметры** по таблице платформы:
   - Размеры визуала (соответствуют ratio?).
   - Длина текста (в лимите?).
   - Формат файла (поддерживается?).
   - Хэштеги (в лимите? нет banned?).
   - Длительность видео (в лимите?).
2. Для каждого ❌ — указать severity и suggested fix.

### Шаг 3 — Правила сообщества
1. Проверить контент на соответствие **правилам платформы**.
2. Проверить **авторские права**: изображения, музыка, видео — лицензия/разрешение.
3. Если paid — проверить **рекламные политики** платформы.
4. Проверить **маркировку рекламы** (если применимо).

### Шаг 4 — Алгоритмические риски
1. Проверить контент на **алгоритмические penalty**:
   - Engagement bait?
   - Внешние ссылки в посте?
   - Водяной знак другой платформы?
   - Повторный контент?
2. Зафиксировать риски как 🟡 Risk.

### Шаг 5 — Вердикт
1. Подсчитать Blocker / Risk / Note.
2. Применить порог прохождения.
3. Для каждого нарушения / риска — suggested fix.

## Валидация (Quality Gate)

Проверка считается завершённой если:

- [ ] Платформа определена, требования загружены из реестра
- [ ] Все технические параметры проверены (размеры, лимиты, форматы)
- [ ] Правила сообщества проверены
- [ ] Авторские права проверены (музыка, фото, видео)
- [ ] Маркировка рекламы проверена (если paid / спонсорский)
- [ ] Алгоритмические риски оценены
- [ ] Для каждого ❌ и ⚠️ — severity и suggested fix
- [ ] Вердикт соответствует порогу

## Handoff

| Результат | Маршрут |
|-----------|---------|
| **COMPLIANT** | → `$content-release-gate` |
| **COMPLIANT WITH RISKS** | → `$content-release-gate` с пометкой рисков (пользователь решает) |
| **NON-COMPLIANT** | → обратно Copywriter / Visual Concept с конкретными исправлениями |

При передаче — использовать `$handoff` с приложением полного отчёта.

## Интеграция с другими скилами

| Скил | Взаимодействие |
|------|----------------|
| `$platform-strategy` | **Источник:** платформы, форматы |
| `$platform-visual-specs` | **Комплемент:** visual specs = технические требования к визуалу, этот скил = полная compliance |
| `$content-review-checklist` | **Комплемент:** review checklist проверяет качество, platform compliance — правила платформы |
| `$brand-compliance` | **Комплемент:** brand = бренд, platform = платформа |
| `$content-release-gate` | **Потребитель:** COMPLIANT обязателен для Release Gate |
| `$content-brief` | **Источник:** технические требования из брифа для сверки |

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Один чеклист для всех платформ | Instagram ≠ LinkedIn ≠ TikTok | Реестр требований по каждой платформе |
| Игнорирование алгоритмических рисков | Контент compliant, но охват 0 | Проверять engagement bait, ссылки, водяные знаки |
| Кросс-пост без адаптации | Видео с TikTok на Instagram со стикером — penalty | Адаптировать под каждую платформу |
| Нет проверки авторских прав | Content ID / DMCA strike → блокировка | Всегда проверять лицензии на музыку, фото, видео |
| Рекламный контент без маркировки | Штрафы, удаление, бан | Маркировка обязательна по закону и правилам платформы |
| «Banned hashtag» не проверен | Shadow ban, снижение охвата | Проверять хэштеги перед публикацией |
| Редактирование поста сразу после публикации | На LinkedIn/Instagram — сброс алгоритмического буста | Всё проверить ДО публикации |
