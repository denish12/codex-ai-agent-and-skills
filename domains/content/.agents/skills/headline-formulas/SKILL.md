# Headline Formulas — Формулы заголовков

## Когда использовать
- При написании **заголовков постов**, статей, email subject lines, рекламных объявлений.
- При создании **hook'ов** для reels, видео, каруселей (первый кадр / первые секунды).
- При **оптимизации существующих** заголовков — низкий CTR, низкий open rate.
- Как часть **`$content-brief`** — секция «Заголовок / Hook».

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Тема / ключевой инсайт | ✅ | О чём контент — главная мысль или факт |
| Персона ЦА | ✅ | Из `$audience-analysis` — язык, боли, триггеры |
| Платформа | ✅ | Определяет лимит символов и стиль |
| Тип контента | ✅ | Пост / статья / email / reel / карусель / лендинг / объявление |
| Этап воронки | ✅ | TOFU / MOFU / BOFU — определяет подход |
| SEO-ключ | ⬚ | Если SEO-контент — основное ключевое слово для title |
| Текущий заголовок | ⬚ | Если оптимизация — текущий вариант и его метрики |

> Если персона или платформа не указаны — **запросить**. Заголовок без ЦА = заголовок для никого.

## Формулы заголовков

### Базовые (универсальные)

| # | Формула | Шаблон | Пример | Лучше всего для |
|---|---------|--------|--------|-----------------|
| 1 | **Число + Результат** | «[N] способов [достичь результата]» | «7 способов удвоить конверсию email» | Статьи, карусели, listicle |
| 2 | **How-to** | «Как [достичь цели] без [проблемы]» | «Как получить первых 1000 подписчиков без бюджета» | Образовательный, TOFU/MOFU |
| 3 | **Вопрос + Боль** | «Почему [проблема]? (и как это исправить)» | «Почему ваши посты не набирают охват?» | Посты, email, nurturing |
| 4 | **Противоречие** | «[Мнение] — на самом деле [факт]» | «Все говорят постить каждый день — вот почему это не работает» | Посты, вирусный, TOFU |

### Продвинутые (высокий CTR)

| # | Формула | Шаблон | Пример | Лучше всего для |
|---|---------|--------|--------|-----------------|
| 5 | **4U** | Urgent + Unique + Ultra-specific + Useful | «До пятницы: 3 шага к первой продаже через Telegram» | Promo, BOFU, email |
| 6 | **Before/After** | «[Было] → [Стало] за [время/действие]» | «0 подписчиков → 10K за 3 месяца: что сработало» | Кейсы, MOFU |
| 7 | **Social Proof** | «[N людей/компаний] уже [результат]» | «500+ маркетологов уже используют эту стратегию» | MOFU/BOFU, promo |
| 8 | **Негатив / Страх** | «[N] ошибок, которые [убивают результат]» | «5 ошибок, которые убивают ваш email open rate» | Образовательный, TOFU |
| 9 | **Curiosity Gap** | «[Интрига] — [подсказка без раскрытия]» | «Мы нашли то, что маркетологи игнорируют годами» | TOFU, вирусный |
| 10 | **Specificity** | «[Точная цифра] + [конкретный результат] + [условие]» | «Как мы получили 847 лидов за $0.12 каждый через одну карусель» | Кейсы, MOFU |

### Специализированные

| # | Формула | Шаблон | Пример | Лучше всего для |
|---|---------|--------|--------|-----------------|
| 11 | **Celebrity / Authority** | «[Авторитет] рекомендует [метод]» | «Метод Гэри Ви для создания 30 постов из одного видео» | TOFU, образовательный |
| 12 | **This vs That** | «[A] vs [B]: что работает лучше в [контекст]» | «Reels vs Карусели: что даёт больше охвата в 2026» | Сравнения, MOFU |
| 13 | **Secret / Insider** | «[Секрет/приём], о котором [не говорят/мало кто знает]» | «Приём копирайтинга, о котором молчат агентства» | TOFU, curiosity |
| 14 | **Challenge** | «[Результат] за [время]: [вызов]» | «30 reels за 30 дней: что произошло с аккаунтом» | Серии, engagement |
| 15 | **Direct Benefit** | «[Получи/Узнай/Научись] [результат] [прямо сейчас]» | «Получи готовый контент-план на апрель — бесплатно» | BOFU, лид-магниты |

## Формулы по этапам воронки

| Этап | Цель заголовка | Лучшие формулы | Стиль |
|------|---------------|----------------|-------|
| **TOFU** | Привлечь внимание, широкий охват | #4 Противоречие, #8 Негатив, #9 Curiosity, #13 Secret, #14 Challenge | Провокационный, интригующий, вирусный |
| **MOFU** | Углубить интерес, создать доверие | #2 How-to, #3 Вопрос+Боль, #6 Before/After, #7 Social Proof, #10 Specificity, #12 This vs That | Экспертный, конкретный, с данными |
| **BOFU** | Конвертировать | #5 4U, #7 Social Proof, #15 Direct Benefit | Прямой, с urgency, с ценностью |

## Power Words

Слова-усилители, которые повышают CTR заголовков:

| Категория | Слова |
|-----------|-------|
| **Ценность** | бесплатно, готовый, пошаговый, проверенный, гарантированный |
| **Urgency** | сейчас, сегодня, до [дата], последний шанс, пока не поздно |
| **Curiosity** | секретный, неожиданный, скрытый, мало кто знает, на самом деле |
| **Негатив** | ошибка, убивает, теряете, никогда, перестаньте, хватит |
| **Эмоция** | невероятный, шокирующий, удивительный, мощный |
| **Конкретика** | точный, пошаговый, конкретный, [число], за [время] |
| **Эксклюзив** | только для, впервые, инсайдерский, закрытый |

> Использовать 1-2 Power Words на заголовок. Больше — clickbait.

## Платформенная адаптация

| Платформа | Лимит символов | Стиль | Особенности |
|-----------|---------------|-------|-------------|
| **Instagram (caption hook)** | Первые 125 символов видны без «ещё» | Короткий, интригующий, эмоциональный | Hook = первая строка. Emoji допустимы. Caps Lock для акцента |
| **Instagram (carousel cover)** | 5-8 слов на обложке | Крупный текст, контрастный | Читаемость > креативность. Один крючок |
| **Telegram** | Первые 2-3 строки видны в превью | Информативный, конкретный | Без clickbait — аудитория Telegram не прощает |
| **LinkedIn** | Первые 150 символов до «...see more» | Профессиональный, экспертный | Начать с инсайта или цифры. Без emoji |
| **Email subject** | ≤ 50 символов (мобильные) | Персональный, curiosity или value | Прехедер дополняет, не повторяет |
| **Google Title (SEO)** | ≤ 60 символов | SEO-ключ в начале + ценность | Ключевое слово как можно раньше |
| **YouTube** | ≤ 70 символов (обрезается ~60) | Curiosity + specificity | Caps Lock для 1-2 слов, числа, эмоция |
| **Twitter/X** | ≤ 280 (но короче = лучше) | Острый, провокационный, мемный | Без ссылок в тексте (снижает охват) |
| **Рекламное объявление** | Зависит от платформы | Прямая выгода + urgency | Тестировать 3-5 вариантов |

## Scorecard — оценка вариантов

Каждый вариант заголовка оценивается по 6 критериям (1-5 баллов):

| Критерий | 1 (слабо) | 3 (средне) | 5 (сильно) |
|----------|-----------|------------|------------|
| **Ясность** | Непонятно о чём | Понятно при вдумчивом чтении | Мгновенно понятно |
| **Конкретность** | Абстрактно («улучшите маркетинг») | Есть намёк на конкретику | Точные цифры, сроки, результаты |
| **Ценность** | Нет выгоды для читателя | Выгода подразумевается | Явная выгода в заголовке |
| **Эмоция / Curiosity** | Скучно, нет реакции | Слегка интересно | Вызывает «серьёзно?!» или «хочу знать» |
| **Соответствие ЦА** | Для всех = для никого | Подходит широко | Говорит на языке конкретной персоны |
| **Платформенность** | Не подходит формату | Подходит, но не оптимально | Идеально для платформы (лимиты, стиль) |

**Порог:** ≥ 22 из 30 — сильный заголовок. 18-21 — доработать. < 18 — переписать.

## Протокол

### Шаг 1 — Контекст
1. Получить входные данные: тема, персона, платформа, воронка, тип контента.
2. Определить лимит символов по платформе.
3. Если SEO — зафиксировать ключевое слово.

### Шаг 2 — Выбор формул
1. По этапу воронки — выбрать **3-4 подходящие формулы** из таблицы.
2. Учесть тип контента (кейс → Before/After, listicle → Число+Результат).
3. Учесть персону (зумеры → провокация, B2B → экспертность).

### Шаг 3 — Генерация
1. Для каждой формулы — написать **1-2 варианта** заголовка.
2. Использовать **1-2 Power Words** (не больше).
3. Проверить лимит символов платформы.
4. Итого: **5-8 вариантов**.

### Шаг 4 — Оценка по Scorecard
1. Оценить каждый вариант по 6 критериям (1-5).
2. Подсчитать итоговый балл.
3. Отсеять варианты < 18 баллов.

### Шаг 5 — Финальный выбор
1. **Основной** — вариант с наивысшим баллом.
2. **A/B тест** — вариант с другой формулой (чтобы тестировать подход, не формулировку).
3. Обосновать выбор.

## Валидация (Quality Gate)

Заголовки считаются готовыми если:

- [ ] Сгенерировано ≥ 5 вариантов по ≥ 3 формулам
- [ ] Каждый вариант в лимите символов платформы
- [ ] Каждый вариант оценён по Scorecard
- [ ] Основной вариант ≥ 22 баллов
- [ ] Выбран вариант для A/B теста (другая формула)
- [ ] Заголовок соответствует этапу воронки
- [ ] Заголовок не обманывает ожидания (не clickbait)
- [ ] Power Words ≤ 2 на заголовок

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Clickbait без содержания | Кликают, но уходят → ER падает, доверие теряется | Заголовок обещает — контент выполняет |
| «Подробнее» / «Новый пост» | Нулевая ценность, никто не кликнет | Конкретная выгода или интрига |
| 5+ Power Words | Выглядит как спам: «БЕСПЛАТНЫЙ СЕКРЕТНЫЙ НЕВЕРОЯТНЫЙ МЕТОД» | Максимум 1-2 Power Words |
| Заголовок не соответствует воронке | TOFU-аудитория видит «Купи сейчас» → скролят мимо | Формула по этапу воронки |
| Один вариант без A/B | Нет данных для оптимизации | Минимум 2 варианта с разными формулами |
| Длинный заголовок на мобильном | Обрезается, теряется смысл | Проверить лимит платформы |
| Абстракция без конкретики | «Улучшите ваш маркетинг» — скучно | Числа, сроки, результаты: «+340% охвата за 2 недели» |
| Заголовок для себя, не для ЦА | Использует профессиональный жаргон для новичков | Язык персоны из `$audience-analysis` |
