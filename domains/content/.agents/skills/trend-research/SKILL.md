# Trend Research — Исследование трендов

## Когда использовать
- При **планировании контента** — какие тренды встроить в `$content-calendar`.
- При **создании ситуативного контента** — тренд прямо сейчас, нужна быстрая реакция.
- При **регулярном мониторинге** — еженедельная проверка трендового поля.
- При **выходе на новую платформу** — что сейчас работает на этой платформе.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Ниша / тема | ✅ | В какой области ищем тренды |
| Платформы | ✅ | Из `$platform-strategy` — где мониторим |
| Персона ЦА | ✅ | Из `$audience-analysis` — что резонирует с аудиторией |
| Бренд-гайдлайны | ✅ | Из `$brand-guidelines` — для brand fit оценки |
| Период | ⬚ | 7 / 14 / 30 дней (по умолчанию 14) |
| Конкуренты | ⬚ | Из `$competitor-content-analysis` — какие тренды используют конкуренты |

> Если ниша не указана — **запросить**. Тренд «в целом» не существует — тренды нише-специфичны.

## Типы трендов

| Тип | Описание | Пример | Скорость жизни |
|-----|----------|--------|---------------|
| **Аудио / Sound** | Вирусный звук, музыка, мем-аудио | Trending sound в TikTok/Reels | 3-14 дней |
| **Формат** | Новый тип контента / подача | Photo dumps, «POV:», text-on-screen templates | 2-8 недель |
| **Тематический** | Горячая тема, дискуссия, новость | AI hype, «тихое увольнение», market crash | 1-4 недели |
| **Визуальный** | Стиль, фильтр, эстетика | AI-портреты, film grain, Y2K aesthetic | 4-12 недель |
| **Challenge** | Челлендж, вызов, марафон | Ice bucket, flip the switch, transition trends | 1-3 недели |
| **Мем** | Вирусный шаблон / формат мема | Distracted boyfriend, «Nobody: ...», Drake | 3-14 дней |
| **Сезонный** | Привязка к времени года, праздникам | Back to school, Black Friday, новогодний | Предсказуемый, ежегодный |

## Жизненный цикл тренда

```
Популярность
     │
     │              ╱ Mainstream
     │             ╱  (все используют)
     │            ╱
     │           ╱ Growth
     │          ╱  (набирает)
     │         ╱
     │        ╱
     │  Early╱                    ╲ Decline
     │      ╱                      ╲ (угасает)
     │     ╱                        ╲
     │    ╱                          ╲ Dead
     └──╱────────────────────────────────── Время
     Emerging   Rising   Peak   Falling   Over
```

| Стадия | Описание | Действие бренда | Скорость реакции |
|--------|----------|----------------|-----------------|
| **Emerging** | Замечен у innovators, ещё нет массового использования | Идеально: быть первым, высокий engagement | 24-48 часов |
| **Rising** | Растёт, early adopters используют | Хорошо: ещё не перенасыщен | 3-7 дней |
| **Peak** | Mainstream, все используют | Ок: если есть уникальный угол. Без угла = шум | 1-3 дня (или пропустить) |
| **Falling** | Пик пройден, вовлечённость падает | Не рекомендуется: выглядит отстало | — |
| **Over** | Мёртвый тренд | Запрещено: репутационный риск | — |

> **Правило:** Лучше пропустить тренд, чем опоздать. Emerging/Rising = зелёная зона. Peak = с оговорками. Falling = нет.

## Brand Fit — подходит ли тренд бренду?

Перед использованием тренда — проверить по 5 критериям:

| Критерий | Вопрос | 🟢 Подходит | 🔴 Не подходит |
|----------|--------|------------|---------------|
| **Ценности** | Тренд совместим с ценностями бренда? | Тренд про productivity → бренд про эффективность | Тренд про «тихое увольнение» → бренд HR-платформа |
| **Аудитория** | Наша ЦА участвует в этом тренде? | Зумеры + TikTok trend → наша ЦА зумеры | B2B CEO + TikTok dance → несовпадение |
| **Платформа** | Тренд на нашей приоритетной платформе? | Reels trend → мы активны в Instagram | TikTok trend → у нас нет TikTok |
| **Tone** | Тренд совместим с ToV бренда? | Юмористический тренд → бренд с игривым тоном | Мемный тренд → серьёзный B2B бренд |
| **Адаптируемость** | Можно ли встроить продукт/тему органично? | Тренд про организацию → наш продукт = планнер | Тренд про еду → мы продаём SaaS |

**Scoring:** 5/5 🟢 = обязательно. 4/5 = да, с адаптацией. 3/5 = осторожно. ≤ 2/5 = пропустить.

## Протокол

### Шаг 1 — Подготовка
1. Определить нишу, платформы, персону.
2. Определить период мониторинга (7/14/30 дней).
3. Выбрать инструменты мониторинга по платформам.

### Шаг 2 — Сбор трендов
1. По каждой платформе — проверить trending-секции, хэштеги, Explore/For You.
2. По кроссплатформенным инструментам — Google Trends, BuzzSumo, Exploding Topics.
3. Проверить, что используют конкуренты.
4. Зафиксировать каждый тренд: название, тип, платформа, стадия жизненного цикла.

### Шаг 3 — Классификация
Для каждого тренда:
1. Определить **тип** (аудио, формат, тематический, визуальный, challenge, мем, сезонный).
2. Определить **стадию** (Emerging → Rising → Peak → Falling → Over).
3. Оценить **метрики** (views, engagement, частота использования).

### Шаг 4 — Brand Fit
Для трендов стадий Emerging / Rising / Peak:
1. Пройти **5 критериев brand fit**.
2. Подсчитать score.
3. Отсеять тренды ≤ 2/5.

### Шаг 5 — Скорость реакции
Для прошедших brand fit:
1. Определить **окно возможности** — сколько времени на адаптацию.
2. Приоритизировать: 24-48 часов = срочно, 3-7 дней = планируемый, 1+ неделя = в календарь.

### Шаг 6 — Адаптация
Для каждого рекомендованного тренда — конкретный план адаптации:
1. **Формат** — как оформить (reel, карусель, пост, thread).
2. **Угол** — как связать тренд с нашей темой/продуктом.
3. **Hook** — как начать (тренд = внимание, бренд = ценность).
4. **CTA** — куда направляем внимание.

### Шаг 7 — Сезонные и запланированные
1. Зафиксировать **предстоящие сезонные тренды** (праздники, события).
2. Добавить в `$content-calendar` заранее (за 2-4 недели).

## Валидация (Quality Gate)

Тренд-исследование считается завершённым если:

- [ ] Платформы промониторены (trending-секции, инструменты)
- [ ] Тренды классифицированы по типу и стадии жизненного цикла
- [ ] Brand fit оценён для каждого тренда (5 критериев)
- [ ] Тренды с fit ≤ 2/5 отсеяны
- [ ] Скорость реакции определена (срочно / планируемый / в календарь)
- [ ] Для рекомендованных — конкретный план адаптации (формат, угол, hook)
- [ ] Угасающие тренды помечены (не использовать)
- [ ] Сезонные тренды зафиксированы для календаря

## Handoff

| Потребитель | Как использует |
|-------------|---------------|
| `$content-calendar` | Тренды → ситуативные слоты в календаре + сезонные заранее |
| `$content-brief` | Конкретный тренд → бриф на единицу контента |
| `$social-media-formats` | Формат адаптации тренда под платформу |
| `$headline-formulas` | Тренд как hook для заголовка |
| Copywriter (агент) | План адаптации → написание текста |
| `$topic-research` | Тематические тренды → глубокое исследование |

При передаче — использовать `$handoff` с рекомендованными трендами и планами адаптации.

## Интеграция с другими скилами

| Скил | Взаимодействие |
|------|----------------|
| `$content-calendar` | **Потребитель:** тренды → ситуативные и сезонные слоты |
| `$content-brief` | **Потребитель:** тренд → конкретный бриф |
| `$competitor-content-analysis` | **Источник:** какие тренды используют конкуренты |
| `$platform-strategy` | **Источник:** платформы для мониторинга |
| `$audience-analysis` | **Источник:** что резонирует с ЦА |
| `$brand-guidelines` | **Источник:** brand fit оценка |
| `$topic-research` | **Комплемент:** trend = что трендит, topic = глубокие факты |

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Опоздать на тренд | Falling/Over тренд = выглядит отстало | Мониторить регулярно, реагировать на Emerging/Rising |
| Тренд ради тренда | Не связан с брендом — аудитория не поймёт | Brand fit ≥ 3/5, органичная связь с темой |
| Серьёзный бренд + мемный тренд | Несовпадение тона → cringe | Проверить Tone fit из brand fit |
| Только один тип тренда | Все reels — одинаковые, нет разнообразия | Комбинировать типы: аудио, формат, тематический |
| Нет адаптации — копирование тренда | «Как все» — нет уникальности, нет бренда | Тренд = формат, бренд = содержание и угол |
| Игнорирование сезонных | Все поздравляют с Новым годом, бренд молчит | Планировать сезонные за 2-4 недели |
| Мониторинг раз в месяц | Тренды живут 3-14 дней — месячный цикл слишком медленный | Мониторинг минимум 1 раз в неделю |
| Использовать controversial тренд | Репутационный риск | Brand fit: ценности + тон. Если сомнения — пропустить |
