# Tone of Voice — Тон голоса бренда

## Когда использовать
- При **создании ToV с нуля** — новый бренд или ребрендинг.
- При **написании любого текста** — копирайтер сверяется с ToV-документом.
- При **онбординге нового копирайтера** — калибровка стиля.
- При **выходе на новую платформу** — адаптация тона.
- Как **развёрнутое дополнение** к секции ToV в `$brand-guidelines`.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Brand Identity | ✅ | Из `$brand-guidelines` — миссия, ценности, архетип, позиционирование |
| Персоны ЦА | ✅ | Из `$audience-analysis` — для адаптации тона по сегментам |
| Платформы | ✅ | Из `$platform-strategy` — для платформенной адаптации |
| Существующий контент | ⬚ | Примеры текстов бренда (если есть) — для анализа текущего тона |
| Конкуренты | ⬚ | Из `$competitor-content-analysis` — для дифференциации тона |

> Если Brand Identity не предоставлена — **запросить** или направить на `$brand-guidelines`. ToV без ядра бренда = случайный стиль.

## Связь с Brand Identity

ToV — это **голосовое выражение** Brand Identity:

| Brand Identity | → | Tone of Voice |
|---------------|---|---------------|
| Миссия | → | О чём говорим (темы, фокус) |
| Ценности | → | Как говорим (принципы коммуникации) |
| Архетип | → | Характер голоса (дружелюбный Опекун vs провокационный Бунтарь) |
| Позиционирование | → | Уровень экспертности и уверенности |

> **Правило:** ToV не может противоречить Brand Identity. Если бренд-архетип «Мудрец» — тон не может быть «мемный и провокационный».

## Протокол

### Шаг 1 — Характеристики голоса
1. Определить **4-6 прилагательных**, описывающих голос бренда.
2. Для каждого — пояснение в 1 предложение: что это значит в контексте бренда.
3. Связать с Brand Identity: какая ценность / элемент архетипа стоит за каждой характеристикой.

### Шаг 2 — Оси тона (7 осей)
Расположить бренд на каждой оси (1-10):

| Ось | 1 | 10 | Что определяет |
|-----|---|------|---------------|
| **Формальность** | Формальный, деловой | Неформальный, разговорный | Регистр языка |
| **Серьёзность** | Серьёзный, сдержанный | Весёлый, игривый | Допустимость юмора |
| **Эмоциональность** | Сдержанный, фактический | Эмоциональный, экспрессивный | Уровень эмоций в тексте |
| **Экспертность** | Доступный, простой | Экспертный, глубокий | Глубина и терминология |
| **Скорость** | Размеренный, обстоятельный | Быстрый, энергичный | Ритм текста, длина предложений |
| **Дистанция** | Отстранённый, нейтральный | Близкий, персональный | «Мы и вы» vs «Привет, друг» |
| **Смелость** | Осторожный, consensus | Провокационный, opinionated | Готовность занять позицию |

### Шаг 3 — Речевые паттерны
Определить конкретные правила:

| Элемент | Варианты | Выбор бренда |
|---------|----------|-------------|
| **Обращение** | Ты / Вы / Мы / Безличное | [выбор + обоснование] |
| **Emoji** | Никогда / Редко (1-2) / Активно | [правило + примеры] |
| **Сленг** | Запрещён / Отраслевой допустим / Молодёжный допустим | [правило] |
| **Длина предложений** | Короткие (5-10 слов) / Средние (10-20) / Длинные (20+) | [предпочтение] |
| **Вопросы читателю** | Не используем / Иногда / Активно | [правило] |
| **Восклицания** | Не используем / Редко / Допустимы | [правило] |
| **Числа** | Словами / Цифрами / Зависит от контекста | [правило] |
| **Аббревиатуры** | Расшифровывать всегда / При первом упоминании / Допустимы без расшифровки | [правило] |
| **Иностранные слова** | Переводить / Транслитерировать / Использовать оригинал | [правило для отрасли] |

### Шаг 4 — «Говорим / Не говорим» по категориям
Для каждой категории — минимум 3 примера:

**Категории:**
1. **Приветствие / обращение** — как начинаем разговор.
2. **Описание продукта** — как говорим о себе.
3. **Описание боли** — как описываем проблемы ЦА.
4. **CTA** — как призываем к действию.
5. **Реакция на успех** — как говорим о результатах.
6. **Реакция на проблему** — как говорим об ошибках, кризисе.
7. **Экспертность** — как показываем знания.

### Шаг 5 — Стоп-лист
Запрещённые слова и конструкции:

| Категория | Запрещено | Почему | Замена |
|-----------|-----------|--------|--------|
| Канцеляризмы | «Осуществлять», «в рамках», «данный» | Мёртвый язык | «Делать», «в», «этот» |
| Корпоративные клише | «Инновационный», «синергия», «экосистема» | Пустые слова | Конкретика: что именно инновационного |
| Манипуляции | «Не упустите свой шанс!!!» | Давление, недоверие | «Попробуйте — вам понравится» |
| [Бренд-специфичные] | [слова, не подходящие бренду] | [обоснование] | [замены] |

### Шаг 6 — Адаптация по персонам
Если ЦА включает разные сегменты — тон может адаптироваться:

| Персона | Адаптация тона | Пример |
|---------|---------------|--------|
| [Персона 1 — новички] | Проще, теплее, без жаргона | «Давай разберёмся вместе, как это работает» |
| [Персона 2 — эксперты] | Глубже, конкретнее, жаргон допустим | «Снизили CAC на 30% через когортный ретаргетинг» |
| [Персона 3 — CEO] | Увереннее, данные, без воды | «ROI 340% за Q1. Вот как мы это сделали» |

> **Правило:** Адаптируется **интенсивность** тона, не сам тон. Бренд остаётся узнаваемым.

### Шаг 7 — Адаптация по платформам

| Платформа | Формальность (сдвиг от базы) | Emoji | Длина | Особенности |
|-----------|------------------------------|-------|-------|-------------|
| **LinkedIn** | +1-2 к формальности | Минимум (0-1) | Средне-длинный | Экспертный, данные, инсайты |
| **Instagram** | -1-2 от формальности | Допустимы (1-3) | Короткий | Визуальный, вдохновляющий, hooks |
| **TikTok** | -2-3 от формальности | Допустимы | Очень короткий | Разговорный, authentic, тренды |
| **Telegram** | База | Минимум | Средний | Информативный, прямой, форматированный |
| **Email** | +1 к формальности | По ситуации | Средний | Персональный, тёплый, чёткий CTA |
| **YouTube** | -1 от формальности | В описании | Разговорный | Как будто говоришь с другом |
| **Блог** | База | Нет | Длинный | Экспертный, но не академический |

> Сдвиг ±1-3 от базовой позиции по осям. Не полная смена тона.

### Шаг 8 — Ситуационная адаптация

| Ситуация | Как меняется тон | Пример |
|----------|-----------------|--------|
| **Хорошая новость** | +эмоциональность, +энергия | «Мы сделали это! 10 000 пользователей за первый месяц 🎉» |
| **Образовательный контент** | +экспертность, -эмоциональность | «Вот 5 проверенных шагов для...» |
| **Продающий** | +уверенность, +ценность | «Попробуй бесплатно — и пойми, почему 500+ компаний выбрали нас» |
| **Кризис / ошибка** | +серьёзность, +ответственность, -юмор | «Мы допустили ошибку. Вот что произошло и что мы делаем, чтобы это исправить» |
| **Поддержка / помощь** | +теплота, +дистанция ближе | «Мы рядом. Напиши — разберёмся вместе» |
| **Behind the scenes** | +неформальность, +близость | «А вот так выглядит наш офис в пятницу вечером» |
| **Поздравление / праздник** | +эмоциональность, +теплота | «С Новым годом! Спасибо, что вы с нами» |

> **Правило:** Ситуация сдвигает оси, но не ломает бренд. Даже в кризисе бренд звучит как бренд.

## Валидация (Quality Gate)

ToV считается готовым если:

- [ ] Характеристики голоса определены (4-6 прилагательных с пояснениями)
- [ ] Все 7 осей заполнены с позицией бренда (1-10)
- [ ] Речевые паттерны зафиксированы (обращение, emoji, сленг, длина, вопросы)
- [ ] «Говорим / Не говорим» — минимум по 3 примера в 7 категориях
- [ ] Стоп-лист составлен (с заменами)
- [ ] Адаптация по персонам (если ЦА неоднородна)
- [ ] Адаптация по платформам (для всех приоритетных платформ)
- [ ] Ситуационная адаптация (минимум 5 сценариев)
- [ ] ToV не противоречит `$brand-guidelines` (миссия, ценности, архетип)
- [ ] Согласован с пользователем

## Handoff

Результат этого скила является **эталоном тона** для всех текстов:

| Потребитель | Как использует |
|-------------|---------------|
| `$brand-guidelines` | ToV-секция в гайдлайне — краткая выжимка из этого документа |
| `$brand-compliance` | **Эталон:** сверяет тон текста с ToV-документом |
| `$content-brief` | Тон указан в секции «Стиль» |
| `$content-review-checklist` | Ревьюер проверяет соответствие тону |
| Copywriter (агент) | Следует ToV при написании любого текста |
| `$social-media-formats` | Адаптация тона по платформам |
| `$email-copywriting` | Тон email'ов |
| `$headline-formulas` | Тон заголовков |

При передаче — использовать `$handoff` с приложением ToV-документа.

## Интеграция с другими скилами

| Скил | Взаимодействие |
|------|----------------|
| `$brand-guidelines` | **Источник:** Brand Identity → ToV. **Потребитель:** ToV-выжимка в гайдлайне |
| `$audience-analysis` | **Источник:** персоны → адаптация тона по сегментам |
| `$platform-strategy` | **Источник:** платформы → адаптация тона по каналам |
| `$competitor-content-analysis` | **Источник:** тон конкурентов → дифференциация |
| `$brand-compliance` | **Потребитель:** эталон для проверки тона |
| `$content-review-checklist` | **Потребитель:** ревьюер проверяет тон |
| `$readability-scoring` | **Комплемент:** тон = как говорим, readability = насколько понятно |

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| ToV = 2 прилагательных | «Дружелюбный и профессиональный» — это 90% брендов | 4-6 характеристик + оси + паттерны + примеры |
| ToV без примеров | Абстрактные правила — каждый интерпретирует по-своему | Минимум по 3 примера «говорим / не говорим» |
| Одинаковый тон везде | LinkedIn-тон в TikTok = неестественно | Адаптация по платформам (сдвиг осей) |
| ToV оторван от бренда | Тон не связан с миссией и ценностями | ToV = голосовое выражение Brand Identity |
| ToV без стоп-листа | Копирайтер использует клише бренда | Явный стоп-лист с заменами |
| Один тон для всех персон | CEO и зумер — разная коммуникация | Адаптация интенсивности по персонам |
| Юмор в кризисе | Аудитория воспринимает как неуважение | Ситуационная адаптация: кризис = серьёзность |
| ToV = документ, который никто не читает | Нет практического применения | Короткие карточки для копирайтеров: «Наш тон — это... Мы говорим... Мы не говорим...» |
