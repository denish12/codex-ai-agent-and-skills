# Примеры промптов — Content Domain

## Создание контента
- `/start-content` — Запустить полный пайплайн создания контента
- `Напиши пост для Instagram о [тема]` — Быстрое создание поста
- `Создай контент-календарь на неделю для [бренд]` — Стратегическое планирование

## Редактирование
- `/edit-content` — Отредактировать существующий текст
- `Перепиши заголовок в стиле [тон]` — Изменить тон

## Быстрая публикация
- `/quick-post` — Быстрый пост без полного пайплайна
