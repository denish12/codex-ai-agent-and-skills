# Prompt Examples — Analytics Domain

## Full Pipeline (`/analyze`)
```
/analyze Analyze the food delivery market in Ukraine: key players, market size, trends, opportunities for a new startup.
```

## Quick Insight (`/quick-insight`)
```
/quick-insight Should our SaaS product expand to the German market? Current MRR $50k, 200 clients in CIS region.
```

## Example Questions for Analysis
- What growth strategy is optimal for a B2B SaaS with LTV/CAC = 2.5?
- Evaluate brand X's competitive position in the electric vehicle market.
- Conduct a SWOT analysis for launching a handmade goods marketplace.
- What EdTech trends will be decisive in the next 2 years?
