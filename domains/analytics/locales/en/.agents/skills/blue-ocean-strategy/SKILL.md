# Blue Ocean Strategy — Creating Uncontested Market Space

## When to Use
- When searching for **new market spaces** beyond existing competition.
- When needing to **radically differentiate** from competitors through value innovation.
- When developing a **new product or service** from scratch — without being tied to current industry standards.
- When preparing a **growth strategy** — when the red ocean is exhausted or unprofitable.

## Input

| Field | Required | Description |
|-------|:--------:|-------------|
| Industry / market | ✅ | Current market (red ocean) for analysis |
| Product / service | ✅ | Current offering or new concept |
| Competitors | ✅ | Key players for building the strategy canvas |
| Customer segments | ⬚ | Current customers and their needs |
| Factors of competition | ⬚ | Known parameters on which the industry competes |
| Price range | ⬚ | Current price levels in the industry |

## Data Sources
1. Competitive analysis — offerings, pricing, positioning of key players.
2. Customer research — needs, pain points, jobs-to-be-done.
3. Industry reports — standard factors of competition, industry norms.
4. Non-customers — why they don't buy, barriers to consumption.
5. Adjacent industries — alternative solutions to the same problems.

## Protocol

### Step 0 — Context Preparation
1. Define the current "red ocean" — the industry and its key players.
2. Identify 5-8 factors of competition on which players compete.
3. Collect data on current customers and their satisfaction.
4. Record industry "sacred cows" — assumptions that nobody challenges.

### Step 1 — Strategy Canvas — AS-IS
1. Define axes: horizontal — factors of competition, vertical — offering level (low / medium / high).
2. Build value curves for each key competitor.
3. Build the value curve for the analyzed company / product.
4. Identify clusters: where all players are identical, where there is differentiation.
5. Record: on which factors the industry competes excessively.

### Step 2 — Buyer Utility Map
1. Define 6 stages of the buyer experience cycle: purchase, delivery, use, supplements, maintenance, disposal.
2. For each stage — assess 6 utility levers: productivity, simplicity, convenience, risk, fun, environmental friendliness.
3. Build a 6x6 matrix: identify untapped utility spaces.
4. Determine: where the industry focuses, and where white spaces exist.

### Step 3 — Three Tiers of Non-Customers
1. **First tier — "soon-to-be"**: customers at the edge of the market, using minimally, seeking alternatives.
   - Why are they at the edge? What dissatisfies them?
2. **Second tier — "refusing"**: consciously chose not to use the industry's products.
   - What needs are unmet? What barriers exist?
3. **Third tier — "unexplored"**: never considered the industry's products as an option.
   - What need do they solve differently? What offering would attract them?
4. Estimate the size of each tier and conversion potential.

### Step 4 — ERRC Framework (Eliminate-Reduce-Raise-Create)
1. **Eliminate**: Which factors, taken for granted in the industry, can be completely removed?
   - Factors that create no value for the buyer.
   - Factors that exist by inertia or to imitate competitors.
2. **Reduce**: Which factors can be reduced well below the industry standard?
   - Factors with excessive offering levels (over-engineering).
3. **Raise**: Which factors should be raised well above the industry standard?
   - Factors where the current level doesn't satisfy buyers.
4. **Create**: Which factors, never offered in the industry, should be created?
   - New sources of value from non-customer analysis and Buyer Utility Map.
5. For each action — rationale and expected impact.

### Step 5 — New Value Curve (Strategy Canvas — TO-BE)
1. Build the new value curve incorporating ERRC.
2. Compare with competitors' curves — visible divergence should be apparent.
3. Verify three characteristics of a good strategy:
   - **Focus**: the curve doesn't replicate competitors' shape.
   - **Divergence**: differs on key parameters.
   - **Compelling tagline**: can be described in a single phrase.
4. If the curve is too similar to competitors — return to Step 4.

### Step 6 — Blue Ocean Opportunity Assessment
1. Estimate the market space size (TAM of non-customers).
2. Analyze the economics: cost reduction (E+R) vs value increase (R+C).
3. Determine defensibility: how long before competitors can copy.
4. Assess feasibility: resources, competencies, time.

### Step 7 — Synthesis and Conclusions
1. Formulate the blue ocean strategic proposition.
2. Define the price corridor of the mass.
3. Record the target profitable model.
4. Prepare an implementation roadmap with key milestones.

## Validation (Quality Gate)

- [ ] AS-IS strategy canvas includes at least 3 competitors and 5 factors
- [ ] ERRC grid completed: at least 1 action in each quadrant
- [ ] New value curve visually differs from competitors'
- [ ] Three tiers of non-customers analyzed with size estimates
- [ ] Buyer Utility Map identified at least 2 untapped spaces
- [ ] Three strategy characteristics verified (focus, divergence, tagline)
- [ ] Economic model justified (cost reduction + value increase)
- [ ] Feasibility assessed (resources, competencies, time)

## Handoff
Result -> Strategist / Mediator for integration into the growth strategy, new product business plan, or innovation program.

## Anti-patterns

| Mistake | Why It's Bad | How to Do It Right |
|---------|-------------|-------------------|
| ERRC without strategy canvas | No baseline for comparison | First AS-IS, then ERRC, then TO-BE |
| Focus only on Create | Increases costs without optimization | Balance: Eliminate + Reduce lower costs, Raise + Create increase value |
| Ignoring non-customers | Missing the main source of the blue ocean | All three tiers of non-customers are mandatory |
| New curve = competitor's curve + 1 factor | That's not a blue ocean, it's incremental innovation | The curve must have a fundamentally different shape |
| Unrealizable proposition | A beautiful strategy without resources is useless | Verify feasibility and economics |
| No compelling tagline | Hard to communicate the strategy | If it can't be described in one phrase — focus is lacking |
