# Примеры промптов — Analytics Domain

## Full Pipeline (`/analyze`)
```
/analyze Проанализируй рынок доставки еды в Украине: ключевые игроки, размер рынка, тренды, возможности для нового стартапа.
```

## Quick Insight (`/quick-insight`)
```
/quick-insight Стоит ли нашему SaaS-продукту выходить на рынок Германии? Текущий MRR $50k, 200 клиентов в СНГ.
```

## Примеры вопросов для анализа
- Какая стратегия роста оптимальна для B2B SaaS с LTV/CAC = 2.5?
- Оцени конкурентную позицию бренда X на рынке электромобилей.
- Проведи SWOT-анализ для запуска маркетплейса handmade товаров.
- Какие тренды в EdTech будут определяющими в ближайшие 2 года?
