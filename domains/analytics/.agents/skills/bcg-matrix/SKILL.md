# Матрица BCG — Портфельный анализ бизнес-единиц

## Когда использовать
- При управлении **портфелем продуктов или бизнес-единиц** — распределение ресурсов.
- При принятии решений об **инвестициях, дивестициях или ликвидации** бизнес-направлений.
- При **стратегическом планировании** — балансировка портфеля по фазам жизненного цикла.
- При подготовке к **M&A** — оценка целевых компаний или подразделений.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Список бизнес-единиц | ✅ | Продукты, бренды, подразделения или сегменты |
| Данные о рынке | ✅ | Объём рынка, темпы роста для каждого сегмента |
| Рыночные доли | ✅ | Доля компании и доля крупнейшего конкурента |
| Выручка по единицам | ⬚ | Для визуализации размера пузырьков и расчёта cash flow |
| Маржинальность по единицам | ⬚ | Для расчёта реального cash flow (gross margin или contribution margin) |
| Пороговые значения | ⬚ | Кастомные пороги роста и доли (по умолчанию: рост 10%, доля 1.0) |
| Горизонт | ⬚ | Период анализа (по умолчанию — текущий год + прогноз 2-3 года) |

> Если обязательные поля не предоставлены — **запросить** у пользователя, не генерировать предположения.

## Источники данных
1. Внутренняя отчётность — выручка, маржа, CAPEX по бизнес-единицам.
2. Маркетинговые исследования — доли рынка, объёмы рынков.
3. Отраслевые отчёты — темпы роста сегментов, прогнозы.
4. Конкурентная разведка — доли основных конкурентов.
5. Финансовые модели — денежные потоки, ROI по направлениям.

### Связь с другими скилами
| Скил | Что берём | Когда вызывать |
|------|-----------|----------------|
| `tam-sam-som` | Объём и потенциал рынков для каждой БЕ (шаги 0, 2) | Если нет данных о размере или росте рынка |
| `competitive-analysis` | Доли конкурентов, позиционирование (шаг 1) | Если неизвестна доля крупнейшего конкурента |

| `swot-analysis` | Сильные/слабые стороны БЕ для решений invest/divest (шаг 5) | При неочевидных решениях по Вопросительным Знакам |
| `web-research` | Верификация рыночных данных, трендов | Если данные отсутствуют или устарели (>12 мес) |

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить портфель: перечислить все анализируемые бизнес-единицы (БЕ).
2. Определить релевантный рынок для каждой БЕ.
3. Установить пороговые значения:
   - Рост рынка: граница высокий/низкий (обычно 10% или средний по портфелю).
   - Относительная доля рынка: граница высокая/низкая (обычно 1.0 — паритет с лидером).
   - Для зрелых отраслей (FMCG, промышленность) допустимо снизить порог роста до 5%.
4. Собрать данные: доли рынка, темпы роста, выручка, маржинальность.

> Каждый факт о рынке — с указанием источника и даты. Данные старше 12 месяцев — пометить ⚠️ и верифицировать через `web-research`.

### Шаг 1 — Расчёт относительной доли рынка
1. Для каждой БЕ рассчитать:

   | БЕ | Доля компании | Доля крупнейшего конкурента | **Отн. доля** | Интерпретация |
   |----|:-------------:|:--------------------------:|:-------------:|---------------|
   | [БЕ1] | XX% | XX% | X.X | Лидер / Не лидер |

2. Значение > 1.0: компания — лидер рынка.
3. Значение < 1.0: компания — не лидер.
4. При отсутствии точных данных — использовать экспертную оценку с пометкой 🔮.

### Шаг 2 — Определение темпов роста рынка
1. Для каждой БЕ определить CAGR рынка за последние 3 года **и** прогноз на 3 года.
2. Классифицировать: выше порога = высокий рост, ниже = низкий рост.
3. Зафиксировать тренд: ускорение ↗, стабильность →, замедление ↘.

   | БЕ | CAGR (факт 3Y) | Прогноз (3Y) | Тренд | Классификация |
   |----|:--------------:|:------------:|:-----:|:-------------:|
   | [БЕ1] | XX% | XX% | ↗/→/↘ | Высокий / Низкий |

### Шаг 3 — Классификация по квадрантам
1. **Звёзды (Stars)** — высокий рост + высокая доля:
   - Лидеры растущих рынков, требуют значительных инвестиций.
   - При замедлении роста трансформируются в Дойных Коров.
2. **Дойные Коровы (Cash Cows)** — низкий рост + высокая доля:
   - Генераторы кеша, требуют минимальных инвестиций.
   - Финансируют Звёзды и Вопросительные Знаки.
3. **Вопросительные Знаки (Question Marks)** — высокий рост + низкая доля:
   - Потенциал есть, но требуют крупных инвестиций для роста доли.
   - Ключевое решение: инвестировать агрессивно или выходить.
4. **Собаки (Dogs)** — низкий рост + низкая доля:
   - Минимальная стратегическая ценность.
   - Кандидаты на оптимизацию, продажу или ликвидацию.
   - **Исключение**: Собаки с высокой синергией (общая инфраструктура, комплементарность, якорный клиент) — оценивать отдельно.

### Шаг 4 — Построение матрицы
1. Расположить каждую БЕ на матрице 2×2.
2. Размер пузырька — пропорционален выручке.
3. Добавить стрелки тренда: куда движется БЕ на горизонте 2-3 года.
4. Отметить БЕ на границе квадрантов (±10% от порога) — пометить как «пограничные».

### Шаг 5 — Стратегия и скоринг решений
1. Для каждой БЕ определить стратегическое действие:

   | Квадрант | Базовая стратегия | Нюансы |
   |----------|-------------------|--------|
   | **Звёзды** | Инвестировать для удержания лидерства | Если тренд ↘ — готовить переход в режим Cash Cow |
   | **Дойные Коровы** | Собирать урожай, минимальные инвестиции | Если тренд ↗ (рынок оживает) — рассмотреть ре-инвестиции |
   | **Вопросительные Знаки** | Селективные инвестиции **или** дивестиция | Решение по скорингу (см. ниже) |
   | **Собаки** | Оптимизировать, продать или ликвидировать | Проверить синергию перед решением |

2. **Скоринг для Вопросительных Знаков** (invest vs divest):

   | Критерий | Вес | Шкала | Invest если |
   |----------|:---:|:-----:|-------------|
   | Размер рынка (TAM) | 0.20 | 1-10 | ≥ 7 — крупный рынок |
   | Конкурентная позиция | 0.20 | 1-10 | ≥ 5 — есть шанс на лидерство |
   | Синергия с портфелем | 0.15 | 1-10 | ≥ 6 — общие каналы, технологии |
   | Стоимость достижения лидерства | 0.20 | 1-10 (инверт.) | ≥ 5 — подъёмные инвестиции |
   | Стратегическая важность | 0.15 | 1-10 | ≥ 7 — критично для будущего |
   | Маржинальность целевая | 0.10 | 1-10 | ≥ 6 — окупается |

   `Score = Σ(вес × оценка)`. Порог: **≥ 6.0 → Invest**, **< 6.0 → Divest**.

3. **Скоринг для Собак** (keep vs exit):

   | Критерий | Keep если |
   |----------|----------|
   | Синергия с Коровами/Звёздами | Высокая — общая инфраструктура, каналы |
   | Cash flow | Положительный — не потребляет ресурсы |
   | Барьер выхода | Высокий — контрактные обязательства, репутация |
   | Стратегический якорь | Да — необходим для удержания ключевого клиента |

   Если ни один критерий не выполнен — рекомендовать exit.

### Шаг 6 — Квантификация cash flow портфеля
1. Рассчитать cash flow по каждому квадранту:

   | Квадрант | Выручка | Маржа | Cash generation | Инвестиции (CAPEX + OPEX) | **Net Cash Flow** |
   |----------|:-------:|:-----:|:---------------:|:-------------------------:|:-----------------:|
   | Звёзды | $XXM | XX% | $XXM | -$XXM | -/+$XXM |
   | Дойные Коровы | $XXM | XX% | $XXM | -$XM | +$XXM |
   | Вопросительные Знаки | $XXM | XX% | $XM | -$XXM | -$XXM |
   | Собаки | $XXM | XX% | $XM | -$XM | ±$XM |
   | **Итого** | | | | | **±$XXM** |

2. Проверить баланс: **Net Cash Flow Дойных Коров ≥ Net Cash Drain Звёзд + Вопросительных Знаков**.
3. Если дисбаланс — пересмотреть инвестиции или ускорить дивестиции.

### Шаг 7 — Анализ траекторий и сценарии
1. Для каждой БЕ спрогнозировать миграцию между квадрантами на горизонте 2-3 года:

   | БЕ | Сейчас | Через 2-3 года | Драйвер миграции |
   |----|:------:|:--------------:|------------------|
   | [БЕ1] | ★ Звезда | $ Корова | Рынок замедляется (CAGR XX% → X%) |
   | [БЕ2] | ? Вопрос | ★ Звезда / ✕ Собака | Зависит от инвестиций |

2. Определить **критические развилки**: БЕ, чья судьба зависит от решений сейчас.
3. Для топ-3 критических БЕ — 3 сценария:

   | БЕ | Сценарий | Результат через 3 года | Вероятность | Условие |
   |----|----------|:----------------------:|:-----------:|---------|
   | [БЕ2] | Invest aggressively | ★ Звезда, доля 1.3 | 40% | +$XM инвестиций, рынок растёт ≥ XX% |
   | [БЕ2] | Maintain current | ? Вопрос → ✕ Собака | 45% | Доля стагнирует, конкуренты растут |
   | [БЕ2] | Divest now | Exit с $XM | 15% | Если покупатель найден |

### Шаг 8 — Синтез и рекомендации
1. Сформулировать портфельную стратегию: общее направление и логику.
2. Определить ТОП-приоритеты по инвестициям и дивестициям.
3. Зафиксировать риски текущей конфигурации портфеля.
4. Определить периодичность пересмотра (ежегодно или при существенном изменении рынка).
5. Зафиксировать go/no-go gates для каждого решения invest/divest.

## Пример — IT-холдинг с 5 продуктами

**Контекст:** IT-холдинг, 5 продуктов. Порог роста: 10%, порог доли: 1.0. Общая выручка $85M.

| БЕ | Рынок | Доля | Доля лидера | Отн. доля | CAGR | Выручка | Квадрант |
|----|-------|:----:|:-----------:|:---------:|:----:|:-------:|:--------:|
| CRM-платформа | CRM для SMB | 28% | 22% | **1.27** | 8% | $30M | $ Корова |
| Облачная аналитика | BI/Analytics | 15% | 35% | **0.43** | 22% | $18M | ? Вопрос |
| Кибербезопасность | SMB Security | 32% | 25% | **1.28** | 18% | $20M | ★ Звезда |
| Корп. мессенджер | Team Comms | 5% | 40% | **0.13** | 4% | $7M | ✕ Собака |
| AI-ассистент | AI Productivity | 12% | 18% | **0.67** | 35% | $10M | ? Вопрос |

**Скоринг Question Marks:**

| Критерий | Облачная аналитика | AI-ассистент |
|----------|:------------------:|:------------:|
| Размер рынка | 7 | 9 |
| Конкурентная позиция | 4 | 6 |
| Синергия | 8 (+ CRM, + Security) | 7 (+ CRM) |
| Стоимость лидерства (инв.) | 3 (дорого, лидер далеко) | 6 (рынок фрагментирован) |
| Стратегическая важность | 6 | 9 |
| Маржинальность | 7 | 5 |
| **Score** | **5.5 → Divest** | **7.0 → Invest** |

**Скоринг Собаки — Корп. мессенджер:**
- Синергия: низкая (отдельная инфраструктура) — нет
- Cash flow: слабо положительный (+$0.3M) — слабое «да»
- Барьер выхода: низкий — нет
- Стратегический якорь: нет → **Рекомендация: exit (продажа)**

**Cash flow баланс:**
- Коровы генерируют: +$12M → Звезда потребляет -$6M, AI-ассистент -$4M → Баланс: +$2M ✅

**Решения:**
1. CRM-платформа (Корова): harvest, направить $12M на Звезду и AI
2. Кибербезопасность (Звезда): invest $6M, удерживать лидерство
3. AI-ассистент (Вопрос → Invest): $4M, цель — доля 1.0 за 2 года
4. Облачная аналитика (Вопрос → Divest): продать или партнёрство
5. Корп. мессенджер (Собака → Exit): продать, ожидаемая цена $5-8M

## Валидация (Quality Gate)

- [ ] Все бизнес-единицы классифицированы по квадрантам с числовыми данными
- [ ] Относительная доля рынка рассчитана корректно (доля / доля крупнейшего конкурента)
- [ ] Темпы роста рынка подкреплены данными (CAGR факт + прогноз)
- [ ] Пороговые значения зафиксированы и обоснованы для отрасли
- [ ] Вопросительные Знаки оценены по скоринговой модели (invest vs divest)
- [ ] Собаки проверены на синергию перед решением об exit
- [ ] Cash flow портфеля квантифицирован: Коровы покрывают Звёзды + Question Marks
- [ ] Траектории миграции определены для каждой БЕ на горизонте 2-3 года
- [ ] Сценарный анализ проведён для топ-3 критических БЕ
- [ ] Тренды и динамика учтены (стрелки ↗/→/↘)
- [ ] Источники данных указаны; данные старше 12 мес помечены ⚠️
- [ ] Go/no-go gates определены для каждого решения invest/divest

## Handoff
Результат передаётся:
- **Стратег / Медиатор** — для формирования портфельной стратегии и бюджетирования.
- **`ansoff-matrix`** — для проработки стратегий роста Звёзд и инвестируемых Вопросительных Знаков.
- **`unit-economics`** — для детального расчёта экономики по каждой БЕ.
- **`competitive-analysis`** — для углублённого анализа конкурентов в квадрантах с низкой долей.

Формат передачи: заполненный шаблон вывода + скоринг Question Marks + cash flow таблица + сценарный анализ.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Использование абсолютной доли вместо относительной | Не показывает позицию vs лидер | Относительная = доля / доля крупнейшего конкурента |
| Неправильные границы квадрантов | Все БЕ оказываются в одном квадранте | Калибровать пороги под отрасль (зрелая = 5%, растущая = 10-15%) |
| Игнорирование размера БЕ | Мелкая «Звезда» и крупная «Собака» неравнозначны | Размер пузырька = выручка, cash flow по квадрантам |
| Статичный анализ без трендов | Не видно, куда движется БЕ — решения на основе snapshot | Стрелки миграции + сценарный анализ на 2-3 года |
| Автоматическая ликвидация Собак | Некоторые Собаки создают синергию или удерживают клиентов | Скоринг keep/exit: синергия, cash flow, барьер выхода, якорь |
| Отсутствие баланса портфеля | Решения по каждой БЕ изолированно — нет системы | Cash flow портфеля: Коровы должны финансировать Звёзды + QM |
| Интуитивное invest/divest для Question Marks | Самый сложный квадрант — интуиция не масштабируется | Скоринговая модель с 6 критериями и порогом ≥ 6.0 |
| Данные без источников и дат | Невозможно верифицировать, быстро устаревают | Каждый факт — с источником и датой; >12 мес = ⚠️ |
