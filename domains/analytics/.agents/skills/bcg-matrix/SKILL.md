# Матрица BCG — Портфельный анализ бизнес-единиц

## Когда использовать
- При управлении **портфелем продуктов или бизнес-единиц** — распределение ресурсов.
- При принятии решений об **инвестициях, дивестициях или ликвидации** бизнес-направлений.
- При **стратегическом планировании** — балансировка портфеля по фазам жизненного цикла.
- При подготовке к **M&A** — оценка целевых компаний или подразделений.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Список бизнес-единиц | ✅ | Продукты, бренды, подразделения или сегменты |
| Данные о рынке | ✅ | Объём рынка, темпы роста для каждого сегмента |
| Рыночные доли | ✅ | Доля компании и доля крупнейшего конкурента |
| Выручка по единицам | ⬚ | Для визуализации размера пузырьков |
| Пороговые значения | ⬚ | Кастомные пороги роста и доли (по умолчанию: рост 10%, доля 1.0) |
| Горизонт | ⬚ | Период анализа (по умолчанию — текущий год) |

## Источники данных
1. Внутренняя отчётность — выручка, маржа, CAPEX по бизнес-единицам.
2. Маркетинговые исследования — доли рынка, объёмы рынков.
3. Отраслевые отчёты — темпы роста сегментов, прогнозы.
4. Конкурентная разведка — доли основных конкурентов.
5. Финансовые модели — денежные потоки, ROI по направлениям.

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить портфель: перечислить все анализируемые бизнес-единицы (БЕ).
2. Определить релевантный рынок для каждой БЕ.
3. Установить пороговые значения:
   - Рост рынка: граница высокий/низкий (обычно 10% или средний по портфелю).
   - Относительная доля рынка: граница высокая/низкая (обычно 1.0 — паритет с лидером).
4. Собрать данные: доли рынка, темпы роста, выручка.

### Шаг 1 — Расчёт относительной доли рынка
1. Для каждой БЕ: относительная доля = доля компании / доля крупнейшего конкурента.
2. Значение > 1.0: компания — лидер рынка.
3. Значение < 1.0: компания — не лидер.
4. Зафиксировать абсолютную долю для контекста.
5. При отсутствии точных данных — использовать экспертную оценку с пометкой.

### Шаг 2 — Определение темпов роста рынка
1. Для каждой БЕ: определить CAGR рынка за последние 3 года или прогноз на 3 года.
2. Классифицировать: выше порога = высокий рост, ниже = низкий рост.
3. Учесть специфику: для зрелых отраслей порог может быть ниже (5%).
4. Зафиксировать прогнозный тренд: ускорение или замедление роста.

### Шаг 3 — Классификация по квадрантам
1. **Звёзды (Stars)** — высокий рост + высокая доля:
   - Лидеры растущих рынков, требуют значительных инвестиций.
   - При замедлении роста трансформируются в Дойных Коров.
2. **Дойные Коровы (Cash Cows)** — низкий рост + высокая доля:
   - Генераторы кеша, требуют минимальных инвестиций.
   - Финансируют Звёзды и Вопросительные Знаки.
3. **Вопросительные Знаки (Question Marks)** — высокий рост + низкая доля:
   - Потенциал есть, но требуют крупных инвестиций для роста доли.
   - Ключевое решение: инвестировать агрессивно или выходить.
4. **Собаки (Dogs)** — низкий рост + низкая доля:
   - Минимальная стратегическая ценность.
   - Кандидаты на оптимизацию, продажу или ликвидацию.

### Шаг 4 — Построение матрицы
1. Расположить каждую БЕ на матрице 2x2.
2. Размер пузырька — пропорционален выручке (если данные доступны).
3. Добавить стрелки тренда: куда движется БЕ (рост/падение доли и рынка).
4. Отметить БЕ, находящиеся на границе квадрантов.

### Шаг 5 — Стратегия распределения ресурсов
1. Для каждой БЕ определить стратегическое действие:
   - **Звёзды**: Инвестировать для удержания лидерства.
   - **Дойные Коровы**: Собирать урожай, минимальные инвестиции.
   - **Вопросительные Знаки**: Селективные инвестиции или дивестиция.
   - **Собаки**: Оптимизировать, продать или ликвидировать.
2. Определить объём перераспределения ресурсов между квадрантами.
3. Рассчитать cash flow по квадрантам: кто генерирует, кто потребляет.

### Шаг 6 — Баланс портфеля
1. Оценить сбалансированность портфеля:
   - Здоровый портфель: Дойные Коровы финансируют Звёзды и Вопросительные Знаки.
   - Дисбаланс: слишком много Собак или отсутствие Звёзд.
2. Определить портфельные GAP: чего не хватает для устойчивого роста.
3. Сформулировать рекомендации по ребалансировке.

### Шаг 7 — Синтез и выводы
1. Сформулировать портфельную стратегию: общее направление.
2. Определить ТОП-приоритеты по инвестициям и дивестициям.
3. Зафиксировать риски текущей конфигурации портфеля.
4. Рекомендовать периодичность пересмотра (обычно ежегодно).

## Валидация (Quality Gate)

- [ ] Все бизнес-единицы классифицированы по квадрантам
- [ ] Относительная доля рынка рассчитана корректно (доля / доля лидера)
- [ ] Темпы роста рынка подкреплены данными (CAGR или прогноз)
- [ ] Пороговые значения зафиксированы и обоснованы
- [ ] Каждая БЕ имеет стратегическое действие
- [ ] Баланс портфеля оценён (cash flow между квадрантами)
- [ ] Тренды и динамика учтены (стрелки движения)
- [ ] Рекомендации конкретны и привязаны к квадрантам

## Handoff
Результат -> Стратег / Медиатор для формирования портфельной стратегии, бюджетирования или M&A-решений.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Использование абсолютной доли вместо относительной | Не показывает позицию vs лидер | Относительная = доля / доля крупнейшего конкурента |
| Неправильные границы квадрантов | Все БЕ оказываются в одном квадранте | Калибровать пороги под отрасль и портфель |
| Игнорирование размера БЕ | Мелкая «Звезда» и крупная «Собака» неравнозначны | Использовать размер пузырька = выручка |
| Статичный анализ без трендов | Не видно, куда движется БЕ | Добавлять стрелки и прогноз на 2-3 года |
| Автоматическая ликвидация «Собак» | Некоторые «Собаки» создают синергию | Проверять стратегическую ценность перед решением |
| Отсутствие баланса портфеля | Решения по каждой БЕ изолированно | Смотреть на портфель целиком: кто финансирует кого |
