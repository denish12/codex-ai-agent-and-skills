# 5 сил Портера — Анализ конкурентных сил отрасли

## Когда использовать
- При оценке **привлекательности отрасли** для входа или инвестиций.
- При разработке **конкурентной стратегии** — понимание расстановки сил на рынке.
- При анализе **изменений в отрасли** — как новые тренды меняют баланс сил.
- При подготовке **M&A решений** — оценка конкурентной среды целевой отрасли.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Отрасль / сегмент | ✅ | Чёткое определение анализируемой отрасли и её границ |
| Географический фокус | ✅ | Страна / регион / глобальный рынок |
| Позиция компании | ⬚ | Текущая или планируемая роль в отрасли |
| Ключевые конкуренты | ⬚ | Список основных игроков для контекста |
| Горизонт анализа | ⬚ | Период оценки (по умолчанию — текущее состояние + 2-3 года) |
| Цель анализа | ⬚ | Вход в рынок, стратегия, инвестиции, M&A |

## Источники данных
1. Отраслевые отчёты — IBISWorld, Statista, отраслевые ассоциации.
2. Финансовая отчётность — годовые отчёты игроков, SEC filings.
3. Конкурентная разведка — рыночные доли, ценообразование, продуктовые линейки.
4. Данные о поставщиках и покупателях — концентрация, switching costs.
5. Регуляторные базы — барьеры входа, лицензирование, патенты.

## Протокол

### Шаг 0 — Подготовка контекста
1. Чётко определить границы отрасли (продуктовые, географические, ценовые сегменты).
2. Идентифицировать ключевых игроков: лидеры, претенденты, нишевые.
3. Определить текущую фазу жизненного цикла отрасли (рост, зрелость, спад).
4. Зафиксировать цель анализа и горизонт.

### Шаг 1 — Угроза новых участников (Threat of New Entrants)
1. Оценить барьеры входа:
   - Экономия масштаба — минимальный эффективный масштаб.
   - Капиталоёмкость — объём начальных инвестиций.
   - Доступ к каналам дистрибуции — сложность выхода на полку / платформу.
   - Патенты и интеллектуальная собственность — наличие защиты.
   - Регуляторные барьеры — лицензии, сертификации, разрешения.
   - Эффект обучения — кривая опыта действующих игроков.
2. Оценить вероятность реализации угрозы на горизонте анализа.
3. Проанализировать реакцию действующих игроков на новых участников.
4. Выставить балл: 1 (низкая угроза) — 5 (высокая угроза).

### Шаг 2 — Сила поставщиков (Supplier Power)
1. Оценить концентрацию поставщиков vs количество покупателей.
2. Проанализировать уникальность ресурса / компонента поставщика.
3. Определить switching costs при смене поставщика.
4. Оценить угрозу прямой интеграции (поставщик становится конкурентом).
5. Проанализировать наличие субститутов для ключевых ресурсов.
6. Выставить балл: 1 (слабая сила) — 5 (высокая сила).

### Шаг 3 — Сила покупателей (Buyer Power)
1. Оценить концентрацию покупателей и долю в выручке.
2. Проанализировать switching costs для покупателей.
3. Определить ценовую чувствительность покупателей.
4. Оценить информированность покупателей о ценах и альтернативах.
5. Проанализировать угрозу обратной интеграции (покупатель производит сам).
6. Оценить стандартизированность продукта (commodity vs дифференциация).
7. Выставить балл: 1 (слабая сила) — 5 (высокая сила).

### Шаг 4 — Угроза товаров-заменителей (Threat of Substitutes)
1. Идентифицировать продукты / услуги из других отраслей, решающие ту же проблему.
2. Оценить соотношение цена/качество субститутов vs текущих продуктов.
3. Проанализировать switching costs при переходе на субститут.
4. Оценить склонность покупателей к переключению.
5. Проанализировать технологические тренды, создающие новые субституты.
6. Выставить балл: 1 (низкая угроза) — 5 (высокая угроза).

### Шаг 5 — Внутриотраслевая конкуренция (Industry Rivalry)
1. Оценить количество и размер конкурентов (фрагментация vs концентрация).
2. Проанализировать темпы роста отрасли (в стагнирующих рынках конкуренция жёстче).
3. Оценить уровень дифференциации продуктов.
4. Проанализировать фиксированные затраты и избыточные мощности.
5. Оценить барьеры выхода — причины оставаться в убыточной отрасли.
6. Проанализировать стратегическую важность отрасли для ключевых игроков.
7. Выставить балл: 1 (слабая конкуренция) — 5 (интенсивная конкуренция).

### Шаг 6 — Общая привлекательность отрасли
1. Рассчитать средний балл по 5 силам.
2. Определить доминирующую силу — та, которая сильнее всего влияет на прибыльность.
3. Сформировать общую оценку привлекательности:
   - 1.0-2.0: Высокая привлекательность.
   - 2.1-3.0: Средняя привлекательность.
   - 3.1-4.0: Низкая привлекательность.
   - 4.1-5.0: Непривлекательная отрасль.
4. Проанализировать динамику: как силы изменятся через 2-3 года.

### Шаг 7 — Синтез и выводы
1. Сформулировать стратегические импликации для каждой из 5 сил.
2. Определить стратегические действия для усиления позиции.
3. Зафиксировать ключевые факторы успеха в отрасли.
4. Подготовить рекомендации: входить / инвестировать / защищаться / уходить.

## Валидация (Quality Gate)

- [ ] Все 5 сил проанализированы с конкретными факторами (минимум 3 на каждую)
- [ ] Каждая сила имеет балл от 1 до 5 с обоснованием
- [ ] Факторы подкреплены данными (доли рынка, концентрация, барьеры)
- [ ] Границы отрасли чётко определены
- [ ] Общая оценка привлекательности рассчитана и обоснована
- [ ] Стратегические рекомендации конкретны и привязаны к силам
- [ ] Учтена динамика сил на горизонте анализа
- [ ] Доминирующая сила идентифицирована

## Handoff
Результат -> Стратег / Медиатор для формирования конкурентной стратегии, инвестиционного решения или стратегии входа на рынок.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Слишком широкие границы отрасли | «Технологическая отрасль» — это 100 разных рынков | Конкретный сегмент: «SaaS CRM для SMB в Европе» |
| Анализ сил без баллов | Описание без оценки не даёт приоритетов | Каждая сила — балл 1-5 с обоснованием |
| Игнорирование динамики | Статичный анализ устаревает | Оценить тренд каждой силы на горизонте 2-3 года |
| Путаница «конкуренты» и «субституты» | Uber и такси — конкуренты; Zoom и авиакомпании — субституты | Субституты — другая отрасль, та же потребность |
| Отсутствие стратегических импликаций | Анализ ради анализа | Каждая сила -> стратегическое действие |
| Одинаковые баллы для всех сил | Нет дифференциации | Использовать полную шкалу, находить доминирующую силу |
