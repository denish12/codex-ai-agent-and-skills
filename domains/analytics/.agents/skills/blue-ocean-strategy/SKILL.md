# Стратегия голубого океана — Создание незанятого рыночного пространства

## Когда использовать
- При поиске **новых рыночных пространств** за пределами существующей конкуренции.
- При необходимости **радикально дифференцироваться** от конкурентов через инновацию ценности.
- При разработке **нового продукта или сервиса** с нуля — без привязки к текущим отраслевым стандартам.
- При подготовке **стратегии роста** — когда красный океан исчерпан или нерентабелен.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Отрасль / рынок | ✅ | Текущий рынок (красный океан) для анализа |
| Продукт / сервис | ✅ | Текущее предложение или концепция нового |
| Конкуренты | ✅ | Ключевые игроки для построения стратегической канвы |
| Клиентские сегменты | ⬚ | Текущие клиенты и их потребности |
| Факторы конкуренции | ⬚ | Известные параметры, по которым конкурируют в отрасли |
| Ценовой диапазон | ⬚ | Текущие ценовые уровни в отрасли |

## Источники данных
1. Конкурентный анализ — предложения, цены, позиционирование ключевых игроков.
2. Клиентские исследования — потребности, pain points, jobs-to-be-done.
3. Отраслевые отчёты — стандартные факторы конкуренции, отраслевые нормы.
4. Не-клиенты — почему не покупают, барьеры входа в потребление.
5. Смежные отрасли — альтернативные решения тех же задач.

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить текущий «красный океан» — отрасль и её ключевых игроков.
2. Идентифицировать 5-8 факторов конкуренции, по которым игроки соревнуются.
3. Собрать данные о текущих клиентах и их удовлетворённости.
4. Зафиксировать отраслевые «священные коровы» — допущения, которые никто не оспаривает.

### Шаг 1 — Стратегическая канва (Strategy Canvas) — AS-IS
1. Определить оси: горизонтальная — факторы конкуренции, вертикальная — уровень предложения (низкий / средний / высокий).
2. Построить кривые ценности для каждого ключевого конкурента.
3. Построить кривую ценности для анализируемой компании / продукта.
4. Выявить кластеры: где все игроки одинаковы, где есть дифференциация.
5. Зафиксировать: по каким факторам отрасль конкурирует избыточно.

### Шаг 2 — Карта полезности для покупателя (Buyer Utility Map)
1. Определить 6 стадий покупательского цикла: покупка, доставка, использование, дополнения, обслуживание, утилизация.
2. Для каждой стадии — оценить 6 рычагов полезности: производительность, простота, удобство, риск, развлечение, экологичность.
3. Построить матрицу 6x6: выявить неиспользованные пространства полезности.
4. Определить: где отрасль фокусируется, а где — белые пятна.

### Шаг 3 — Три уровня не-клиентов
1. **Первый уровень — «скоро уйдут»**: клиенты на границе рынка, используют минимально, ищут альтернативы.
   - Почему они на границе? Что их не устраивает?
2. **Второй уровень — «отказники»**: сознательно выбрали не пользоваться продуктами отрасли.
   - Какие потребности не удовлетворены? Какие барьеры?
3. **Третий уровень — «неисследованные»**: никогда не рассматривали продукты отрасли как вариант.
   - Какую потребность они решают иначе? Какое предложение их привлечёт?
4. Оценить размер каждого уровня и потенциал конверсии.

### Шаг 4 — Фреймворк ERRC (Eliminate-Reduce-Raise-Create)
1. **Eliminate (Упразднить)**: Какие факторы, принятые в отрасли как данность, можно полностью убрать?
   - Факторы, не создающие ценности для покупателя.
   - Факторы, существующие по инерции или для имитации конкурентов.
2. **Reduce (Снизить)**: Какие факторы можно снизить значительно ниже отраслевого стандарта?
   - Факторы с избыточным уровнем предложения (over-engineering).
3. **Raise (Повысить)**: Какие факторы нужно поднять значительно выше отраслевого стандарта?
   - Факторы, где текущий уровень не удовлетворяет покупателей.
4. **Create (Создать)**: Какие факторы, никогда не предлагавшиеся в отрасли, нужно создать?
   - Новые источники ценности из анализа не-клиентов и Buyer Utility Map.
5. Для каждого действия — обоснование и ожидаемый эффект.

### Шаг 5 — Новая кривая ценности (Strategy Canvas — TO-BE)
1. Построить новую кривую ценности с учётом ERRC.
2. Сравнить с кривыми конкурентов — должна быть видимая дивергенция.
3. Проверить три характеристики хорошей стратегии:
   - **Фокус**: кривая не повторяет форму конкурентов.
   - **Дивергенция**: отличается по ключевым параметрам.
   - **Убедительный слоган**: можно описать одной фразой.
4. Если кривая слишком похожа на конкурентов — вернуться к шагу 4.

### Шаг 6 — Оценка возможности голубого океана
1. Оценить размер рыночного пространства (TAM не-клиентов).
2. Проанализировать экономику: снижение затрат (E+R) vs рост ценности (R+C).
3. Определить защитимость: как долго конкуренты не смогут скопировать.
4. Оценить реализуемость: ресурсы, компетенции, время.

### Шаг 7 — Синтез и выводы
1. Сформулировать стратегическое предложение голубого океана.
2. Определить ценовой коридор массы (price corridor of the mass).
3. Зафиксировать целевую прибыльную модель.
4. Подготовить дорожную карту реализации с ключевыми вехами.

## Валидация (Quality Gate)

- [ ] Стратегическая канва AS-IS содержит минимум 3 конкурента и 5 факторов
- [ ] ERRC-сетка заполнена: минимум 1 действие в каждом квадранте
- [ ] Новая кривая ценности визуально отличается от конкурентных
- [ ] Три уровня не-клиентов проанализированы с оценкой размера
- [ ] Buyer Utility Map выявила минимум 2 неиспользованных пространства
- [ ] Три характеристики стратегии проверены (фокус, дивергенция, слоган)
- [ ] Экономическая модель обоснована (снижение затрат + рост ценности)
- [ ] Реализуемость оценена (ресурсы, компетенции, время)

## Handoff
Результат -> Стратег / Медиатор для интеграции в стратегию роста, бизнес-план нового продукта или инновационную программу.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| ERRC без стратегической канвы | Нет базы для сравнения | Сначала AS-IS, потом ERRC, потом TO-BE |
| Фокус только на Create | Повышает затраты без оптимизации | Баланс: Eliminate + Reduce снижают затраты, Raise + Create повышают ценность |
| Игнорирование не-клиентов | Теряется главный источник голубого океана | Все три уровня не-клиентов обязательны |
| Новая кривая = кривая конкурента + 1 фактор | Это не голубой океан, а инкрементальная инновация | Кривая должна иметь фундаментально другую форму |
| Нереализуемое предложение | Красивая стратегия без ресурсов бесполезна | Проверять реализуемость и экономику |
| Отсутствие убедительного слогана | Сложно коммуницировать стратегию | Если нельзя описать одной фразой — не хватает фокуса |
