# Анализ цепочки ценности — Первичные и поддерживающие виды деятельности

## Когда использовать
- При поиске **конкурентных преимуществ** — где компания создаёт больше всего ценности.
- При оптимизации **структуры затрат** — выявление неэффективных звеньев цепочки.
- При принятии решений о **make-or-buy** — аутсорсинг, вертикальная интеграция, партнёрства.
- При **бенчмаркинге** — сравнение цепочки ценности с конкурентами для выявления разрывов.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Организация / БЕ | ✅ | Компания или подразделение для анализа |
| Отрасль | ✅ | Контекст отрасли для бенчмаркинга |
| Продукт / услуга | ⬚ | Конкретный продукт для фокусного анализа |
| Конкуренты для сравнения | ⬚ | 1-3 конкурента для бенчмаркинга |
| Финансовые данные | ⬚ | Затраты по видам деятельности (если доступны) |
| Стратегическая цель | ⬚ | Оптимизация затрат, дифференциация, обе |

## Источники данных
1. Управленческий учёт — затраты по видам деятельности (ABC-костинг).
2. Операционные метрики — производительность, cycle time, качество по звеньям.
3. Данные о клиентах — удовлетворённость, NPS, жалобы по точкам контакта.
4. Конкурентный анализ — бенчмарки по ключевым видам деятельности.
5. Отраслевые стандарты — лучшие практики, средние показатели.

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить объект анализа и границы (вся компания или конкретная БЕ / продукт).
2. Зафиксировать стратегическую цель: лидерство по затратам, дифференциация или и то, и другое.
3. Определить уровень детализации: укрупнённый (9 видов) или детальный (суб-активности).
4. Собрать доступные финансовые и операционные данные.

### Шаг 1 — Первичные виды деятельности: Входящая логистика (Inbound Logistics)
1. Идентифицировать ключевые процессы: приём, хранение, управление запасами, транспортировка от поставщиков.
2. Оценить затраты на входящую логистику (% от общих затрат).
3. Определить добавленную ценность: надёжность поставок, качество входных материалов.
4. Выявить узкие места и неэффективности.
5. Сравнить с конкурентами (если данные доступны).

### Шаг 2 — Первичные виды деятельности: Операции (Operations)
1. Идентифицировать ключевые процессы: производство, сборка, тестирование, упаковка.
2. Оценить затраты на операции (% от общих затрат).
3. Определить добавленную ценность: качество, скорость, гибкость производства.
4. Оценить технологический уровень и степень автоматизации.
5. Выявить источники конкурентного преимущества или отставания.

### Шаг 3 — Первичные виды деятельности: Исходящая логистика (Outbound Logistics)
1. Идентифицировать ключевые процессы: хранение готовой продукции, доставка, дистрибуция.
2. Оценить затраты на исходящую логистику (% от общих затрат).
3. Определить добавленную ценность: скорость доставки, покрытие, надёжность.
4. Проанализировать каналы дистрибуции и их эффективность.
5. Оценить удовлетворённость клиентов доставкой.

### Шаг 4 — Первичные виды деятельности: Маркетинг и продажи (Marketing & Sales)
1. Идентифицировать ключевые процессы: реклама, продвижение, продажи, ценообразование, каналы.
2. Оценить затраты на маркетинг и продажи (% от общих затрат).
3. Определить добавленную ценность: узнаваемость бренда, конверсия, клиентские отношения.
4. Оценить эффективность каналов привлечения (CAC, ROAS, конверсия).
5. Проанализировать ценовое позиционирование и стратегию.

### Шаг 5 — Первичные виды деятельности: Сервис (Service)
1. Идентифицировать ключевые процессы: поддержка, гарантия, обучение, обслуживание.
2. Оценить затраты на сервис (% от общих затрат).
3. Определить добавленную ценность: удовлетворённость, retention, LTV.
4. Оценить качество сервиса: SLA, время ответа, CSAT, NPS.
5. Проанализировать возможности для монетизации сервиса.

### Шаг 6 — Поддерживающие виды деятельности: Инфраструктура (Firm Infrastructure)
1. Проанализировать: управление, планирование, финансы, юридический, качество.
2. Оценить затраты на инфраструктуру (% от общих затрат).
3. Определить вклад в эффективность первичных видов деятельности.
4. Оценить качество управленческих решений и планирования.

### Шаг 7 — Поддерживающие виды деятельности: Управление HR (Human Resource Management)
1. Проанализировать: найм, обучение, развитие, мотивация, удержание.
2. Оценить затраты на HR (% от общих затрат).
3. Определить вклад в качество и производительность первичных видов деятельности.
4. Оценить уровень компетенций и кадровые риски.

### Шаг 8 — Поддерживающие виды деятельности: Технологическое развитие (Technology Development)
1. Проанализировать: R&D, IT, автоматизация, инновации.
2. Оценить затраты на технологическое развитие (% от общих затрат, % от выручки).
3. Определить вклад в дифференциацию и эффективность.
4. Оценить технологический разрыв с конкурентами.

### Шаг 9 — Поддерживающие виды деятельности: Снабжение (Procurement)
1. Проанализировать: закупки, управление поставщиками, контракты.
2. Оценить затраты на снабжение (% от общих затрат).
3. Определить вклад: качество входных ресурсов, условия, надёжность.
4. Оценить переговорную позицию и стратегию закупок.

### Шаг 10 — Анализ затрат и ценности по звеньям
1. Построить диаграмму распределения затрат по всем 9 видам деятельности.
2. Для каждого вида определить: затраты vs добавленная ценность.
3. Выделить виды деятельности с максимальным GAP (высокие затраты, низкая ценность).
4. Выделить виды деятельности с максимальным конкурентным преимуществом.

### Шаг 11 — Синтез и выводы
1. Определить ТОП-3 источника конкурентного преимущества.
2. Определить ТОП-3 зоны для оптимизации.
3. Сформулировать рекомендации: оптимизация, аутсорсинг, инвестиции, интеграция.
4. Подготовить план действий с приоритетами и ожидаемым эффектом.

## Валидация (Quality Gate)

- [ ] Все 5 первичных видов деятельности проанализированы
- [ ] Все 4 поддерживающих вида деятельности проанализированы
- [ ] Для каждого вида указаны затраты (% или абсолютные) и добавленная ценность
- [ ] Узкие места и неэффективности выявлены
- [ ] Конкурентные преимущества идентифицированы с обоснованием
- [ ] Рекомендации конкретны: оптимизация / аутсорсинг / инвестиции / интеграция
- [ ] Бенчмаркинг с конкурентами проведён (хотя бы экспертная оценка)
- [ ] Итоговая таблица затрат/ценности заполнена для всех 9 видов

## Handoff
Результат -> Стратег / Медиатор для формирования операционной стратегии, программы оптимизации или конкурентной стратегии.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Анализ только первичных видов | Поддерживающие часто определяют конкурентное преимущество | Все 9 видов: 5 первичных + 4 поддерживающих |
| Затраты без ценности | Непонятно, оправданы ли затраты | Всегда пара: затраты + добавленная ценность |
| Игнорирование связей между видами | Оптимизация одного звена может ухудшить другое | Анализировать взаимозависимости (логистика <-> операции) |
| Общие формулировки без данных | «Хороший маркетинг» не даёт инсайтов | Конкретика: CAC = $X, конверсия = Y%, ROAS = Z |
| Анализ без стратегического контекста | Оптимизация затрат vs дифференциация — разные фокусы | Привязать к стратегии: что оптимизировать, что усиливать |
| Отсутствие бенчмаркинга | Непонятно, хорошие показатели или плохие | Сравнивать с конкурентами и отраслевыми стандартами |
