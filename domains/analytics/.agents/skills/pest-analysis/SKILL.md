# PEST/PESTEL-анализ — Анализ макросреды

## Когда использовать
- При оценке **макроокружения** перед выходом на новый рынок или запуском нового продукта.
- При **стратегическом планировании** — для понимания внешних факторов, влияющих на бизнес.
- При анализе **регуляторных и политических рисков** в конкретных регионах или отраслях.
- При подготовке **инвестиционных решений** — оценка странового и отраслевого риска.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Объект анализа | ✅ | Компания, отрасль, рынок или регион |
| Географический фокус | ✅ | Страна / регион / глобальный охват |
| Горизонт анализа | ⬚ | Период прогноза (по умолчанию — 3-5 лет) |
| Режим | ⬚ | PEST (4 фактора) или PESTEL (6 факторов, по умолчанию) |
| Отрасль | ⬚ | Конкретная отрасль для фокусировки факторов |
| Стратегические цели | ⬚ | Контекст для оценки влияния факторов |

## Источники данных
1. Государственные статистические агентства — макроэкономические показатели, демография.
2. Центральные банки и регуляторы — ставки, регуляторные инициативы.
3. Отраслевые ассоциации — отраслевые отчёты, прогнозы.
4. Международные организации — World Bank, IMF, WTO, OECD.
5. Технологические обзоры — Gartner, Forrester, McKinsey, BCG.
6. Экологические и правовые базы — ESG-рейтинги, реестры законодательства.

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить объект анализа и географический фокус.
2. Уточнить режим: PEST (4 фактора) или PESTEL (6 факторов).
3. Зафиксировать горизонт анализа и стратегический контекст.
4. Определить ключевые вопросы, на которые должен ответить анализ.
5. Собрать доступные данные из источников.

### Шаг 1 — Политические факторы (Political)
1. Оценить политическую стабильность и форму правления.
2. Проанализировать текущую и планируемую государственную политику, влияющую на отрасль.
3. Оценить торговую политику: тарифы, санкции, торговые соглашения.
4. Проанализировать налоговую политику и государственное стимулирование.
5. Оценить коррупционные риски и качество институтов.
6. Для каждого фактора: определить влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 2 — Экономические факторы (Economic)
1. Проанализировать макроэкономические показатели: ВВП, инфляция, безработица.
2. Оценить валютные риски и динамику курсов.
3. Проанализировать процентные ставки и доступность капитала.
4. Оценить покупательскую способность и структуру потребления.
5. Проанализировать цикличность экономики и фазу делового цикла.
6. Для каждого фактора: влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 3 — Социальные факторы (Social)
1. Проанализировать демографические тренды: возрастная структура, урбанизация, миграция.
2. Оценить культурные нормы и ценности, влияющие на потребление.
3. Проанализировать уровень образования и квалификацию рабочей силы.
4. Оценить изменения в образе жизни и потребительских предпочтениях.
5. Проанализировать социальное неравенство и его влияние на рынки.
6. Для каждого фактора: влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 4 — Технологические факторы (Technological)
1. Оценить уровень технологического развития в регионе / отрасли.
2. Проанализировать темпы технологических изменений и дизрупций.
3. Оценить инвестиции в R&D — государственные и частные.
4. Проанализировать уровень цифровизации и интернет-проникновения.
5. Оценить влияние автоматизации и AI на отрасль.
6. Для каждого фактора: влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 5 — Экологические факторы (Environmental, PESTEL)
1. Проанализировать экологическое законодательство и регуляторные требования.
2. Оценить климатические риски и их влияние на бизнес.
3. Проанализировать ESG-тренды и давление стейкхолдеров.
4. Оценить доступность и стоимость ресурсов (энергия, сырьё, вода).
5. Проанализировать углеродный след и требования к устойчивому развитию.
6. Для каждого фактора: влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 6 — Правовые факторы (Legal, PESTEL)
1. Проанализировать регуляторную среду: лицензирование, сертификация, комплаенс.
2. Оценить защиту интеллектуальной собственности и патентное право.
3. Проанализировать трудовое законодательство и его влияние на кадровую стратегию.
4. Оценить защиту данных и приватности (GDPR, локальные аналоги).
5. Проанализировать антимонопольное регулирование и его влияние на рынок.
6. Для каждого фактора: влияние (1-5), вероятность (1-5), тренд (↑/↓/→).

### Шаг 7 — Scoring и приоритизация
1. Рассчитать итоговый балл для каждого фактора: Влияние x Вероятность.
2. Определить ТОП-факторы по каждой категории (score >= 12).
3. Построить тепловую карту рисков и возможностей.
4. Выделить критические факторы, требующие немедленного реагирования.

### Шаг 8 — Синтез и выводы
1. Сформулировать общий профиль макросреды (благоприятная / нейтральная / враждебная).
2. Определить ключевые драйверы изменений на горизонте анализа.
3. Зафиксировать сценарии: оптимистичный, базовый, пессимистичный.
4. Подготовить стратегические рекомендации с учётом макрофакторов.

## Валидация (Quality Gate)

- [ ] Все категории (P/E/S/T или P/E/S/T/E/L) проработаны с минимум 3 факторами
- [ ] Каждый фактор имеет оценку влияния (1-5), вероятности (1-5) и тренд
- [ ] Факторы подкреплены данными и источниками
- [ ] Scoring корректен: итоговый балл = влияние x вероятность
- [ ] Тренды обоснованы (↑/↓/→) с указанием причин
- [ ] Критические факторы выделены и имеют рекомендации
- [ ] Анализ учитывает заданный горизонт и географический фокус
- [ ] Отсутствуют дублирования между категориями

## Handoff
Результат -> Стратег / Медиатор для интеграции в стратегический анализ, оценку рисков или инвестиционное решение.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Перечисление факторов без оценки влияния | Список без приоритетов бесполезен | Каждый фактор — с баллом влияния и вероятности |
| Игнорирование трендов | Статичный снимок не показывает динамику | Всегда указывать тренд ↑/↓/→ и горизонт |
| Смешение категорий | Правовой фактор в «Политических» путает анализ | Строго разделять по PESTEL-категориям |
| Фокус только на негативных факторах | Теряются возможности | Анализировать и риски, и возможности |
| Отсутствие географической привязки | Факторы различаются по странам | Всегда указывать регион / страну |
| Одинаковые баллы для всех факторов | Нет дифференциации, нет приоритетов | Использовать полную шкалу 1-5, обосновывать |
