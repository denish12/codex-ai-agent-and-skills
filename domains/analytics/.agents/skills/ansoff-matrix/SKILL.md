# Матрица Ансоффа — Стратегия роста бизнеса

## Когда использовать
- При выборе **стратегии роста** — куда направить ресурсы для масштабирования.
- При оценке **новых рыночных возможностей** — выход на новые рынки или сегменты.
- При планировании **расширения продуктовой линейки** — новые продукты для существующих или новых клиентов.
- При принятии решений о **диверсификации** — оценка рисков и потенциала.

## Вход

| Поле | Обязательно | Описание |
|------|:-----------:|----------|
| Компания / бизнес-единица | ✅ | Объект, для которого определяется стратегия роста |
| Текущие продукты | ✅ | Перечень существующих продуктов / услуг |
| Текущие рынки | ✅ | Перечень текущих рынков, сегментов, географий |
| Стратегические цели роста | ⬚ | Целевые показатели (выручка, доля, прибыль) |
| Доступные ресурсы | ⬚ | Бюджет, компетенции, временные рамки |
| Конкурентный контекст | ⬚ | Положение на рынке, сильные стороны, бренд |

## Источники данных
1. Внутренняя отчётность — выручка и маржа по продуктам и рынкам.
2. Маркетинговые исследования — размер рынков, потенциал роста, клиентские сегменты.
3. Конкурентный анализ — доли рынка, стратегии роста конкурентов.
4. Данные о клиентах — LTV, churn, cross-sell потенциал, удовлетворённость.
5. Отраслевые прогнозы — тренды, новые ниши, технологические сдвиги.

## Протокол

### Шаг 0 — Подготовка контекста
1. Определить текущий портфель продуктов и рынков.
2. Зафиксировать текущую позицию: выручка, доля, рентабельность по каждой паре «продукт-рынок».
3. Определить стратегические цели роста и горизонт.
4. Оценить доступные ресурсы и ограничения.
5. Установить риск-аппетит организации (консервативный / умеренный / агрессивный).

### Шаг 1 — Проникновение на рынок (Market Penetration)
1. Проанализировать потенциал увеличения доли на текущих рынках с текущими продуктами.
2. Оценить возможности:
   - Увеличение частоты потребления существующими клиентами.
   - Привлечение клиентов конкурентов (switching).
   - Конверсия не-потребителей в текущем сегменте.
3. Определить инструменты: ценообразование, промо, дистрибуция, лояльность.
4. Оценить потенциальный эффект: прирост выручки / доли.
5. Оценить риск: низкий (знакомый продукт, знакомый рынок).
6. Определить необходимые инвестиции и ROI.

### Шаг 2 — Развитие рынка (Market Development)
1. Идентифицировать новые рынки для существующих продуктов:
   - Новые географии (города, страны, регионы).
   - Новые клиентские сегменты (возраст, доход, B2B vs B2C).
   - Новые каналы дистрибуции (онлайн, партнёры, франшиза).
2. Для каждого нового рынка оценить:
   - Размер и потенциал (TAM, SAM, SOM).
   - Барьеры входа (регуляторные, культурные, конкурентные).
   - Необходимая адаптация продукта (локализация, кастомизация).
3. Оценить риск: средний (знакомый продукт, новый рынок).
4. Определить необходимые инвестиции и ожидаемый ROI.

### Шаг 3 — Развитие продукта (Product Development)
1. Идентифицировать новые продукты для существующих рынков:
   - Расширение текущей продуктовой линейки (новые SKU, тарифы).
   - Апгрейды и следующие поколения продуктов.
   - Комплементарные продукты и сервисы.
   - Кросс-продажи (cross-sell) и апсейлы (up-sell).
2. Для каждого нового продукта оценить:
   - Соответствие потребностям текущих клиентов (product-market fit).
   - Синергия с текущими компетенциями и инфраструктурой.
   - Время до вывода на рынок (time-to-market).
3. Оценить риск: средний (новый продукт, знакомый рынок).
4. Определить необходимые R&D-инвестиции и ожидаемый ROI.

### Шаг 4 — Диверсификация (Diversification)
1. Рассмотреть варианты диверсификации:
   - **Связанная (related)**: новый продукт на новом рынке, но с синергией с текущим бизнесом.
   - **Несвязанная (unrelated)**: полностью новое направление без синергии.
2. Для каждого варианта оценить:
   - Стратегическое обоснование (зачем выходить за рамки текущего бизнеса).
   - Потенциальная синергия (технологии, бренд, дистрибуция, клиенты).
   - Требуемые компетенции — есть ли они внутри или нужна M&A / партнёрство.
   - Размер инвестиций и срок окупаемости.
3. Оценить риск: высокий (новый продукт, новый рынок).
4. Рассмотреть альтернативы: партнёрство, JV, лицензирование vs органический рост.

### Шаг 5 — Оценка рисков по квадрантам
1. Построить матрицу риск/доходность для каждой стратегии.
2. Типичная шкала рисков:
   - Проникновение: 1-2 (низкий).
   - Развитие рынка: 2-3 (средний).
   - Развитие продукта: 2-3 (средний).
   - Диверсификация: 4-5 (высокий).
3. Сопоставить с риск-аппетитом организации.
4. Определить необходимые меры митигации для каждой стратегии.

### Шаг 6 — Рекомендация стратегического пути роста
1. Ранжировать стратегии по: потенциал роста, риск, ROI, реализуемость.
2. Определить оптимальную комбинацию (портфель стратегий роста).
3. Распределить ресурсы между квадрантами (% бюджета).
4. Сформулировать дорожную карту: что делать сначала, что позже.

### Шаг 7 — Синтез и выводы
1. Сформулировать рекомендуемый путь роста.
2. Определить ключевые KPI для отслеживания прогресса.
3. Зафиксировать критические решения и сроки.
4. Подготовить план действий с ответственными и вехами.

## Валидация (Quality Gate)

- [ ] Все 4 квадранта проанализированы с конкретными инициативами
- [ ] Для каждой инициативы определён потенциал, риск и ROI
- [ ] Текущий портфель «продукт-рынок» зафиксирован с метриками
- [ ] Риски оценены и сопоставлены с аппетитом организации
- [ ] Рекомендуемый путь роста обоснован данными
- [ ] Распределение ресурсов между квадрантами определено
- [ ] Дорожная карта имеет конкретные вехи и ответственных
- [ ] Меры митигации рисков зафиксированы

## Handoff
Результат -> Стратег / Медиатор для интеграции в стратегический план, бюджет или бизнес-план.

## Anti-patterns

| Ошибка | Почему плохо | Как правильно |
|--------|-------------|---------------|
| Фокус только на проникновении | Потенциал текущего рынка конечен | Оценивать все 4 квадранта, искать баланс |
| Диверсификация без обоснования | Самый рискованный квадрант без стратегической логики | Диверсификация только при чётком стратегическом обосновании |
| Одна стратегия вместо портфеля | Класть все яйца в одну корзину рискованно | Комбинировать стратегии с разным уровнем риска |
| Игнорирование рисков | Оптимистичные планы без проверки | Каждый квадрант — с оценкой риска и мерами митигации |
| Отсутствие метрик текущей позиции | Рост «от неизвестного» невозможен измерить | Начинать с фиксации baseline: выручка, доля, маржа |
| Смешение развития рынка и продукта | Непонятно, где источник роста | Чётко разделять: новый рынок vs новый продукт |
