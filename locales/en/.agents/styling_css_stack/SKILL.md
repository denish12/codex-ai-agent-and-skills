# Skill: Styling (CSS/Less/Sass/Tailwind)

## Rules
- Tailwind by default (if the project is designed that way)
- If CSS/Less/Sass is used - uniform naming conventions and structure
- UI must match UX/a11y: contrast, focus styles, states
- Avoid “random” global styles that break neighboring components
